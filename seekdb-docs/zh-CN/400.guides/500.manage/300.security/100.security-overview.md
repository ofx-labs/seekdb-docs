# 安全权限概述

数据库系统的重要指标之一是要确保系统安全，通过数据库管理系统防止非授权使用数据库，保护数据库的文件和数据。本文为您介绍 seekdb 的安全体系。

seekdb 作为数据的最终载体，时刻关注保护数据的安全。seekdb 已经支持比较完整的企业级安全特性，包括身份鉴别和认证、访问控制、监控告警，可以有效保证用户的数据安全。

* 身份鉴别和认证

  seekdb 支持身份标识和鉴别、用户管理和角色管理，提升数据库的安全性。详情参见 [身份鉴别和认证](200.identification-and-authentication/100.identification.md)。

* 访问控制

 seekdb 通过定义各种系统、对象权限以及角色来控制用户对数据的访问。详情参见 [访问控制](300.access-control/100.access-control-overview.md)。

* 数据加密

  seekdb 支持在数据传输过程中，对数据进行加密，详情参见 [数据传输加密](400.data-transfer-encryption/100.data-transfer-encryption-overview.md)。
