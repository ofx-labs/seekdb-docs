# 角色管理概述

seekdb 支持用户角色管理功能。

角色是一组权限的组合。通过使用角色，您可以轻松地管理用户权限。

在 seekdb 中，角色具备以下功能：

* 与普通用户一样，角色可以被授予权限。

* 角色可以被授予其他角色，即角色中可以包含其他角色。

* 一个用户可以被授予多个角色，一个角色也可以授予多个用户。

当前，seekdb 暂不支持默认的系统角色。
