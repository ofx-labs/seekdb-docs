# obshell 概述

obshell（OceanBase Shell）是 OceanBase 为运维人员 & 开发人员提供的免安装、开箱即用的本地数据库命令行工具。obshell 支持集群和单机（seekdb）运维，实现了不同生态产品对同一数据库统一管理，从而方便第三方产品工具对接 OceanBase 数据库和 seekdb，同时降低了 OceanBase 数据库 和 seekdb 的管理难度和成本。

obshell 不需要额外安装。默认情况下，您通过任何方式安装 OceanBase seekdb 后都可以在安装目录的 `usr/bin` 目录下看到 obshell 可执行文件。
