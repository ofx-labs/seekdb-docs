# 实例管理

登录 obshell Dashboard 控制台，默认展示 **实例管理** 页面。本文介绍 **实例管理** 页面展示的信息及可执行的操作。

**实例管理** 页面展示 seekdb 的基本信息、白名单信息、实例状态及最近一次合并开始和结束时间。同时支持通过 **实例管理** 页面对 seekdb 进行启停和重启操作、对 obshell 进行升级操作，以及对实例执行合并操作。

## 基本信息

**基本信息** 模块展示了实例的状态、seekdb 版本号、实例规格、包含的数据库和用户数、端口、创建时间、工作目录等信息，具体展示参见下图。

单击 **实例管理** 后的刷新图标将根据实例最新信息刷新 **基本信息** 模块；单击 **数据库数** 或 **用户数** 后的数字，可对应跳转到 **数据库管理** 或 **用户管理** 页面查看数据库或用户信息；单击 **连接字符串** 后的 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-01.png" alt="复制" style={{ height: '20px' }}/></span> 图标可复制连接串以备后续使用。

![基本信息](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-01.png)

## 启停实例

:::info
   obshell Dashboard 目前不支持启停通过 systemd 管理的 seekdb 实例，若当前 seekdb 实例通过 systemd 启动，请执行 <code>systemctl stop seekdb</code>/<code>systemctl start seekdb</code> 命令停止/启动 seekdb 实例。
:::

当实例为 **正常运行** 状态时，单击 **实例管理** 页面右上角 **停止** 按钮，在确认弹窗中单击 **停止**，obshell Dashboard 将提交停止实例的任务并展示任务 ID，您可单击展示的任务 ID 跳转查看任务的执行情况。

:::tip
   停止实例将导致业务中断，请确认相关风险后再执行操作。
:::

![启停实例](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-02.png)

停止实例任务完成后，实例状态将变更为 **已停止**，单击 **实例管理** 页面右上角 **启动** 按钮，在确认弹窗中单击 **启动**，obshell Dashboard 将提交启动实例的任务并展示任务 ID，您可单击展示的任务 ID 跳转查看任务的执行情况。

## 重启实例

当实例为 **正常运行** 状态时，单击 **实例管理** 页面右上角 **重启** 按钮，在确认弹窗中单击 **重启**，obshell Dashboard 将提交重启实例的任务并展示任务 ID，您可单击展示的任务 ID 跳转查看任务的执行情况。

:::tip
   重启实例将导致业务中断，请确认相关风险后再执行操作。
:::

![重启实例](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-03.png)

## 升级 obshell

光标悬停在右上角 **···** 后单击出现的 **升级 obshell** 按钮。

在出现的 **Agent 升级版本** 窗口配置 **obshell Agent 版本**，版本是从当前已上传的 obshell 软件包中选择，若当前 obshell Dashboard 中无高于当前版本的 obshell 软件包，可单击下拉框中的 **+ 上传软件包** 进行上传。

:::info
   您需保证本地存在所需安装包。若本地无对应安装包，可单击 <b>上传软件包</b> 页面的 <b>RPM 包下载地址</b> 字段访问开源镜像站下载对应软件包。
:::

配置完成后单击 **升级**，在弹出的窗口中单击 **确定**，obshell Dashboard 将提交升级 obshell 的任务，您可单击弹窗中的 **查看任务** 按钮查看任务执行情况。

![升级 obshell](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-04.png)

## 白名单

**白名单** 模块展示了能够连接到实例的地址列表。您可单击模块右侧的修改图标修改白名单。

![修改白名单](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-05.png)

支持自定义设置可访问 IP，或设置为所有 IP 都可访问。选择自定义设置时，可在 **自定义 IP** 下的输入框中指定允许登陆的客户端列表，输入 IP 后单击键盘 Enter 键或输入英文逗号（`,`）可配置多个节点 IP。光标悬停在 **查看配置说明** 上可查看配置 IP 的说明，配置 IP 时支持的格式如下：

* IP 地址，示例：127.0.0.10,127.0.0.11
* 子网/掩码，示例：127.0.0.0/24
* 模糊匹配，示例：127.0.0.% 或 127.0.0._
* 多种格式混合，示例：127.0.0.10,127.0.0.2,127.0.0.%,127.0.0._,127.0.0.1/24

:::info
   设置为 <code>%</code> 表示所有客户端都可以连接，存在访问安全风险，请谨慎操作。
:::

## 合并管理

**合并管理** 模块展示了当前实例的 **状态**、**最近 1 次合并开始时间** 和 **最近 1 次合并结束时间**。单击 **状态** 后的 **发起合并**，在弹出的窗口中单击 **确定** 即可发起合并。

![发起合并](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/200.dashboard-instance-06.png)
