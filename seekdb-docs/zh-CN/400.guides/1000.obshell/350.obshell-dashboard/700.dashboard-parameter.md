# 参数管理

登录 obshell Dashboard 控制台，单击左侧导航栏 **参数管理** 即可进入 **参数管理** 页面。本文介绍如何通过 obshell Dashboard 查看并修改 seekdb 实例的参数。

**参数管理** 页面展示了 seekdb 中的 **参数名称**、**当前值**、**说明**，以及可对参数进行的操作。光标悬停在对应参数的 **说明** 列可查看对应参数的详细说明；如需查看指定的配置项，您可在页面右上角搜索框中搜索对应配置项。

![参数总览](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/700.dashboard-parameter-01.png)

单击 **操作** 列的 **修改值** 字段可修改对应配置项的值。修改完成后单击 **提交** 按钮，并进行二次确认后即可完成参数的修改。

![修改参数](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/700.dashboard-parameter-02.png)
