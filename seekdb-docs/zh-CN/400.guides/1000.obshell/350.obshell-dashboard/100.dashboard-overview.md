# obshell Dashboard

obshell Dashboard 是 OceanBase 提供的基于 Web 的可视化管理页面，支持对 seekdb 进行资源管理。它提供了关键资源和性能指标的监控功能，帮助用户高效管理 seekdb。

obshell Dashboard 服务完全集成在 obshell 进程中，用户可以通过访问任何部署了 obshell 的服务地址来管理 seekdb。这种设计不仅简化了管理流程，还确保了极低的资源开销，为用户提供简单易用的管控体验。

## 特性介绍

obshell Dashboard 目前支持 seekdb 管理的以下特性：

* 实例管理：查看实例信息，包含状态信息、部署规格、创建/启动时间、工作目录等核心信息，并提供启停实例、修改实例白名单等功能，具体介绍可参见 [实例管理](200.dashboard-instance.md)。

* 数据库管理：管理实例下的数据库，具体介绍可参见 [数据库管理](300.dashboard-db.md)。

* 用户管理：管理实例下的用户，具体介绍可参见 [用户管理](400.dashboard-user.md)。

* 告警中心：查看实例的告警规则和告警事件，并支持对告警进行屏蔽。具体介绍可参见 [告警管理](500.dashboard-alarm.md)。

* 性能监控：实时监控 seekdb 实例的 **性能与SQL**、**事务**、**存储与缓存**，以及 **主机性能** 指标，具体介绍可参见 [性能监控](600.dashboard-monitor.md)。

* 参数管理：查看及修改 seekdb 实例的参数，具体介绍可参见 [参数管理](700.dashboard-parameter.md)。

* 软件包管理：上传软件包，升级 obshell 需保证已上传目标版本的软件包。具体操作可参见 [软件包管理](800.dashboard-package.md)。

* 任务中心：查看及管理实例运行过程中产生的任务，具体介绍可参见 [任务管理](900.ob-dashboard-task.md)。

## 登录与退出

obshell 支持通过图形化界面可视化管理 seekdb，在成功安装 obshell 后，使用 obshell 节点 IP 和 obshell 端口即可访问 obshell Dashboard（obshell 可视化管理页面）。访问时默认使用 root 用户登录，密码为实例中 root 用户密码。若 root 用户密码为空，可直接单击 **登录** 按钮登陆。

登录 obshell Dashboard 控制台后，光标悬停在页面右上角 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-overview-01.png" alt="用户" style={{ height: '20px' }}/></span> 图标处，单击出现的 **退出登录**，在弹出的窗口单击 **确定** 即可退出登录。

## 查看版本

登录 obshell Dashboard 控制台后，单击页面右上角 **帮助** > **关于 obshell Dashboard**，即可查看 obshell Dashboard 的版本。
