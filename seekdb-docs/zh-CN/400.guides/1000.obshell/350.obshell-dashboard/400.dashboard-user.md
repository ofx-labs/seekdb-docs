# 用户管理

登录 obshell Dashboard 控制台，在左侧导航栏中单击 **用户管理**，即可进入 **用户管理** 页面。

**用户管理** 页面展示了当前实例包含的所有用户，您可单击 **登录连接串** 列连接串后的 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-01.png" alt="复制" style={{ height: '20px' }}/></span> 图标复制对应用户连接串以备后续使用。若用户过多，您可在页面右上角搜索栏中输入用户名搜索查看指定用户信息。

![概览](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-01.png)

## 新建用户

单击 **用户管理** 页面右上角 **新建用户** 按钮，在 **新建数据库用户** 页面中配置 **用户名**、**密码**、**全局权限** 和 **数据库权限**。设置完成后单击 **提交** 即完成新建。

![新建用户](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-02.png)

| 参数  | 说明  |
|-------|------|
| 用户名 | 设置新建用户的名称，用户名需满足如下要求：<ul><li>长度为 2 ~ 64 个字符。</li><li>可包含小写英文字母、数字和下划线，但需以英文字母开头。</li></ul> |
| 密码 | 密码复杂度需满足如下要求：<ul><li>长度为 8 ~ 32 个字符</li><li>至少包含以下四种类型字符各 2 个：数字（0 ~ 9）、大写字母（A ~ Z）、小写字母（a ~ z）和特殊字符（<code>~!@#%^&*_-+=\|(){}[]:;,.?/</code>）</li></ul> 配置后可单击 **复制密码**，复制后保存已配置的密码以备后续使用。 |
| 确认密码  | 再次输入配置的密码进行二次确认，若两次输入的密码不一致将提示 `两次输入的新密码不一致，请重新输入`。 |
| 全局权限（可选） | 可通过下拉框选择用户可拥有的权限，设置的权限适用于所有的数据库。各权限的介绍可参见 [MySQL 模式下的权限分类](https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000003980139)，未配置时默认该用户无任何权限。  |
| 数据库权限 | 设置待新建的用户对每个数据库的权限，适用于指定数据库中的所有目标。设置方法如下：<ol><li>勾选左侧列表框中的一个或多个数据库。左侧列表框中显示了当前实例中所有可选的数据库，若数据库过多也可通过左侧列表框中的搜索框搜索指定数据库。</li><li> 单击中间的 **>** 按钮。此时勾选的数据库会被移动到右侧列表框。</li><li>在右侧列表框中勾选设置指定数据库中用户可拥有的权限，勾选对应数据库右上角的 **全部** 可为用户配置当前数据库的所有权限。</li><li>重复 1、2、3 动作，为不同的数据库设置用户权限，若移至右侧列表框中的数据库过多，也可在右侧搜索框中搜索指定数据库。</li><li>如需取消指定数据库的用户权限，可在右侧列表框中单击数据库右上角的 **删除** 将对应数据库移回左侧列表框。您也可单击右侧列表框右上角的 **移除全部** 字段，在弹出的窗口中单击 **确定** 后将右侧列表框中的所有数据库移回左侧列表框。</li></ol>  |

## 锁定用户

可通过 **锁定** 列下的开关图标控制是否锁定用户，当用户处于未锁定状态时，单击 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/400.ob-dashboard-tenant-user-01.png" alt="开关" style={{ height: '20px' }}/></span>，在弹出的窗口中单击 **确定** 即可锁定当前用户，被锁定的用户将不允许登录，请谨慎操作。当用户被锁定时，单击 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/400.ob-dashboard-tenant-user-02.png" alt="开关" style={{ height: '20px' }}/></span>，在弹出的窗口中单击 **确定** 即可解锁当前用户，解锁用户将允许其登录。

![锁定用户](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-03.png)

## 修改密码

单击 **操作** 列下的 **修改密码** 字段，在弹出的窗口中填写 **新密码** 并 **确认密码**，填写完成后单击 **确定** 即可完成对用户密码的修改。密码复杂度需满足如下要求：

* 长度为 8 ~ 32 个字符。

* 至少包含以下四种类型字符各 2 个：数字（0 ~ 9）、大写字母（A ~ Z）、小写字母（a ~ z）和特殊字符（<code>~!@#%^&*_-+=\|(){}[]:;,.?/</code>）。

配置后可单击 **复制密码**，复制后保存已配置的密码以备后续使用。

![修改密码](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-04.png)

## 修改权限

单击 **操作** 列下的 **修改权限** 字段可修改对应用户的全局权限和数据库权限。全局权限可直接通过下拉框修改，数据库权限的修改方法如下：

1. 勾选左侧列表框中的一个或多个数据库。左侧列表框中显示了当前实例中所有可选的数据库，若数据库过多也可通过左侧搜索框搜索指定数据库。

2. 单击中间的 **>** 按钮。此时勾选的数据库会被移动到右侧列表框。

3. 在右侧列表框中勾选设置指定数据库中用户可拥有的权限，勾选对应数据库右上角的 **全部** 可为用户配置当前数据库的所有权限。

4. 重复 1、2、3 动作，为不同的数据库设置用户权限，若移至右侧列表框中的数据库过多，也可在右侧搜索框中搜索指定数据库。

5. 如需取消指定数据库的用户权限，可在右侧列表框中单击数据库右上角的 **删除** 将对应数据库移回左侧列表框。您也可单击右侧列表框右上角的 **移除全部** 字段，在弹出的窗口中单击 **确定** 后将右侧列表框中的所有数据库移回左侧列表框。

6. 设置完成后单击 **提交** 即可完成修改。

![修改权限](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-05.png)

## 删除用户

单击 **操作** 列下的 **删除** 字段，在弹出的窗口输入栏中输入 `delete` 后单击 **删除** 按钮即可删除对应用户。删除用户将删除该用户下所有对象和数据，请谨慎操作。

![删除用户](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/400.dashboard-user-06.png)
