# 任务管理

**任务中心** 页面可查看 obshell Dashboard 中的所有任务，本文介绍 **任务中心** 页面信息及可进行的操作。

您可通过如下两个入口访问 **任务中心** 页面：

* 入口一：登录 obshell Dashboard 控制台后，单击页面右上角 **任务** 字段即可进入 **任务中心** 页面。

* 入口二：登录 obshell Dashboard 控制台后，单击左侧导航栏 **系统管理** > **任务中心** 进入 **任务中心** 页面。

您可通过任务状态筛选任务，或通过搜索框查看指定任务，单击对应任务名即可查看任务详情。

![任务中心](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/900.ob-dashboard-task-01.png)

任务处于 **运行中** 状态时，可单击对应任务名进入任务详情后，单击页面右上角 **取消任务** 按钮，二次确认后取消整个任务。

若任务处于 **失败** 状态，对于失败任务，您可选择执行如下操作：

* 单击 **定位当前进度** 按钮，页面将跳转到失败子任务页面，根据任务详情页面右侧展示的日志信息进行排查。

* 单击 **回滚任务** 按钮，在确认窗口中单击 **回滚**。回滚操作将从失败处开始回滚所有已执行过的任务。

* 单击 **跳过任务** 按钮，在确认窗口中单击 **跳过**。跳过操作将跳过整个任务。

* 单击 **重试** 按钮，在确认窗口中单击 **确认**。重试操作会重试所有失败节点，并从失败节点开始继续向下执行。

* 单击右侧日志信息页面右上角的 **下载日志**，下载当前环节日志访问官网 [问答区](https://open.oceanbase.com/) 发帖咨询，我们的值班工程师将协助您解决问题。
