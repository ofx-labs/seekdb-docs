# 数据库管理

登录 obshell Dashboard 控制台，在左侧导航栏中单击 **数据库管理**，即可进入 **数据库管理** 页面。本文介绍 **数据库管理** 页面的信息及该页面可进行的操作。

## 基本信息

**数据库管理** 页面展示实例中包含数据库的 **数据库名**、**字符集**、**Collation**、**JDBC 连接串**、**创建时间**、是否 **只读**，以及可执行的操作。您可单击 **JDBC 连接串** 列连接串后的 <span style={{ verticalAlign: 'middle' }}><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-01.png" alt="复制" style={{ height: '20px' }}/></span> 图标复制对应数据库的连接串以备后续使用。

您可单击 **创建时间** 后的排序图标，按照升序或降序的顺序展示数据库；存在多个数据库的情况下，如需查找数据库，也可在页面右上角搜索栏中输入数据库名搜索查看指定数据库信息。

![概览](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/300.dashboard-db-01.png)

## 新建数据库

单击 **数据库管理** 页面右上角 **新建数据库** 按钮，在 **新建数据库** 弹窗中配置 **数据库名**、**字符集**、**Collation** 以及是否只读。设置完成后单击 **提交** 即完成新建。

![新建数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/300.dashboard-db-02.png)

| 参数  | 说明  |
|-------|------|
| 数据库名 | 设置新建数据库的名称，数据库名需满足如下要求：<ul><li>长度为 2 ~ 128 个字符。</li><li>可包含小写英文字母、数字和下划线，但需以英文字母开头。</li></ul> |
| 字符集 | 默认为 `utf8mb4`，可通过下拉框选择字符集。 |
| Collation  | 可通过下拉框选择合适的 Collation。 |
| 只读 | 勾选设置待新建数据库是否为只读数据库。  |

## 修改数据库

单击 **操作** 列下的 **编辑** 字段即可修改对应数据库的 **字符集**、**Collation** 和是否只读。修改内容后单击 **提交** 即可完成对数据库的修改。

![编辑数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/300.dashboard-db-03.png)

## 删除数据库

单击 **操作** 列下的 **删除** 字段，在弹出的窗口输入栏中输入 `delete` 后单击 **删除** 按钮即可删除对应数据库。

![删除数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/SeekDB/obshell/300.dashboard-db-04.png)
