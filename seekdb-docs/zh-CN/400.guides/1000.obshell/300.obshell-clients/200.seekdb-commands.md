# obshell seekdb 命令组

本文介绍 obshell 的 seekdb 命令组，seekdb 命令组可用于管理 seekdb。您可通过在命令中配置 `-h`/`--help` 选项查看命令的帮助信息，如 `obshell seekdb start -h`；也可在命令中配置 `-v`/`--verbose` 查看命令的详细执行过程。

:::info
   seekdb 命令组专为管理 seekdb 实例设计，执行该组命令时无需显式指定 <code>--seekdb</code> 选项。
:::

## obshell seekdb start

使用该命令可启动 seekdb。

```shell
obshell seekdb start [--port] [--seekdb] [-6]

# example
obshell seekdb start --port 2886
```

选项说明见下表：

| 选项名 | 是否必选 | 数据类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| --port | 否 | int | 2886 | 指定 obshell 的端口号。若不想通过该选项指定端口，也可通过环境变量 `OBSHELL_PORT_FOR_SEEKDB` 指定 obshell 端口。 |
| --seekdb | 否 | 无 | 无 | 该选项无需传值，配置即表示命令应用于 seekdb。 |
|  -6/--use-ipv6 | 否 | 无 | 无 | 该选项无需传值，配置即表示使用 IPv6。  |

## obshell seekdb restart

使用该命令可重启 seekdb。

```shell
obshell seekdb restart [-t] [--port] [--seekdb] [-6] [-y]

# example
obshell seekdb restart -t --port 2886
```

选项说明见下表：

| 选项名 | 是否必选 | 数据类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| -t/--terminate | 否 | 无 | 无 | 该选项无需传值，命令中配置即表示在重启 seekdb 之前，触发 `MINOR FREEZE` 命令。 |
| --port | 否 | int | 2886 | 指定 obshell 的端口号。若不想通过该选项指定端口，也可通过环境变量 `OBSHELL_PORT_FOR_SEEKDB` 指定 obshell 端口。 |
| --seekdb | 否 | 无 | 无 | 该选项无需传值，配置即表示命令应用于 seekdb。 |
|  -6/--use-ipv6 | 否 | 无 | 无 | 该选项无需传值，配置即表示使用 IPv6。  |
| -y/--yes  | 否 | 无 | 无 | 该选项无需传值，命令中配置即表示跳过重启操作的二次确认。 |

## obshell seekdb stop

使用该命令可停止 seekdb。

```shell
obshell seekdb stop [-t] [--port] [--seekdb] [-6] [-y]

# example
obshell agent stop -t --port 2886
```

选项说明见下表：

| 选项名 | 是否必选 | 数据类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| -t/--terminate | 否 | 无 | 无 | 该选项无需传值，命令中配置即表示在停止 seekdb 进程之前，触发 `MINOR FREEZE` 命令。 |
| --port | 否 | int | 2886 | 指定 obshell 的端口号。若不想通过该选项指定端口，也可通过环境变量 `OBSHELL_PORT_FOR_SEEKDB` 指定 obshell 端口。 |
| --seekdb | 否 | 无 | 无 | 该选项无需传值，配置即表示命令应用于 seekdb。 |
|  -6/--use-ipv6 | 否 | 无 | 无 | 该选项无需传值，配置即表示使用 IPv6。  |
| -y/--yes  | 否 | 无 | 无 | 该选项无需传值，命令中配置即表示跳过停止操作的二次确认。 |

## obshell seekdb show

使用该命令可查看 seekdb 的相关配置及状态。

```shell
obshell seekdb show [-d] [--port] [--seekdb] [-6]

# example
obshell agent show
```

选项说明见下表：

| 选项名 | 是否必选 | 数据类型 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| -d/--show_detail | 否 | 无 | 无 | 该选项无需传值，配置即表示展示 seekdb 的详细信息。 |
| --port | 否 | int | 2886 | 指定 obshell 的端口号。若不想通过该选项指定端口，也可通过环境变量 `OBSHELL_PORT_FOR_SEEKDB` 指定 obshell 端口。 |
| --seekdb | 否 | 无 | 无 | 该选项无需传值，配置即表示命令应用于 seekdb。 |
|  -6/--use-ipv6 | 否 | 无 | 无 | 该选项无需传值，配置即表示使用 IPv6。  |
