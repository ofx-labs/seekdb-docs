# 创建用户

在企业用户，部署 seekdb 时，推荐使用普通用户，要求用户具备 sudo 权限和作为数据库相关目录的所有者。本文以创建 admin 用户为例，介绍如何创建用户并为用户设置 sudo 权限。

1. 执行以下命令，创建用户组 admin 以及创建用户 admin。

   ```shell
   useradd -U admin -d /home/admin -s /bin/bash
   chown -R admin:admin /home/admin
   ```

2. 执行以下命令，为账户 admin 设置密码。

   ```shell
   passwd admin
   ```

3. 为账户 admin 设置 sudo 权限。

   执行以下命令，打开 `/etc/sudoers` 文件：

   ```shell
   vim /etc/sudoers
   ```

   在 `/etc/sudoers` 文件末尾添加如下内容：

   ```shell
   ## Same thing without a password
   # %wheel        ALL=(ALL)       NOPASSWD: ALL
   admin       ALL=(ALL)       NOPASSWD: ALL
   ```
