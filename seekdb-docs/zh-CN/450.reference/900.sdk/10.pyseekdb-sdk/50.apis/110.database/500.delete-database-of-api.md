# delete_database - 删除数据库

`delete_database()` 用于删除一个数据库。

:::info

仅支持在使用 `AdminClient` 连接时，才能使用该接口。关于 `AdminClient` 的详细介绍，参见 [Admin Client](../100.admin-client.md)。

:::

## 前提条件

* 您已经安装了 pyseekdb，有关安装 pyseekdb 的详细信息，参见 [快速开始](../../../10.pyseekdb-sdk/10.pyseekdb-sdk-get-started.md)。

* 您已经连接到数据库。有关连接的详细操作参见 [Admin Client](../100.admin-client.md)。

* 如果您使用的是服务器模式的 seekdb 或者 OceanBase 数据库，请确保连接的用户具备 `DROP` 权限。查看当前用户权限的相关操作请参见 [查看用户权限](https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000003980135)。如果不具备该权限，请联系管理员为您授权，用户授权的相关操作请参见 [直接授予权限](https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000003980140)。

## 请求参数

```python
delete_database(name,tenant=DEFAULT_TENANT)
```

|参数|取值类型|是否必选|描述|取值示例|
|---|---|---|---|---|
|`name`|string|必选|指定要删除的数据库的名称。|my_database|
|`tenant`|string|可选<ul><li>当使用嵌入式 seekdb 和服务器模式 seekdb 时，不需要填写该参数。</li><li>当使用 OceanBase 数据库时，该参数必选。默认值为 `sys`。</li></ul>|指定所删除的数据库归属的租户。|test_tenant|

## 请求示例

```python
import pyseekdb

# Embedded mode
admin = pyseekdb.AdminClient(path="./seekdb")

# Delete database
admin.delete_database("my_database")
```

## 返回参数

无
