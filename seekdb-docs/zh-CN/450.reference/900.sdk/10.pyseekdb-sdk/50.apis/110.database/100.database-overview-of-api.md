# 数据库管理

数据库（Database）下包括若干表、索引，以及数据库对象的元数据信息。用户可以根据自己的需求创建、查询、删除数据库。

对于 collection 的操作，支持使用以下 API 接口。

| API 接口|描述     |相关文档链接|
|---|---|---|
| `create_database()` | 创建数据库。 | [文档](200.create-database-of-api.md) |
| `get_database()` | 获取指定数据库。 |[文档](300.get-database-of-api.md)|
| `list_databases()`  | 获取实例中数据库列表。  |[文档](400.list-database-of-api.md)|
| `delete_database()`  | 删除指定数据库。|[文档](500.delete-database-of-api.md)|
