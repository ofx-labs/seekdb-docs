# DML 操作

DML（数据操作语言）操作允许您在 collection 中插入、更新和删除数据。

对于 DML 的操作，支持使用以下 API 接口。

| API 接口|描述     |相关文档链接|
|---|---|---|
| `add()` | 将新记录插入到 collection 中。 | [文档](200.add-data-of-api.md) |
| `update()` | 更新 collection 中的现有记录。 |[文档](300.update-data-of-api.md)|
| `upsert()`  | 插入新记录或更新现有记录。  |[文档](400.upsert-data-of-api.md)|
| `delete()`  | 从 collection 中删除记录。|[文档](500.delete-data-of-api.md)|
