# API 参考

seekdb 支持通过 API 以编程的方式来使用 seekdb。

## API 接口

目前支持以下 API 接口。

### Database

:::info
仅支持在使用 `AdminClient` 连接时，才能使用该接口。关于 `AdminClient` 的详细介绍，参见 [Admin Client](../50.apis/100.admin-client.md)。
:::

| API 接口|描述     |相关文档链接|
|---|---|---|
| `create_database()` | 创建数据库。 | [文档](110.database/200.create-database-of-api.md) |
| `get_database()` | 获取指定数据库。 |[文档](110.database/300.get-database-of-api.md)|
| `list_databases()`  | 获取实例中数据库列表。  |[文档](110.database/400.list-database-of-api.md)|
| `delete_database()`  | 删除指定数据库。|[文档](110.database/500.delete-database-of-api.md)|

### Collection

:::info
仅支持在使用 `Client` 连接时，才能使用该接口。关于 `Client` 的详细介绍，参见 [Client](../50.apis/50.client.md)。
:::

| API 接口|描述     |相关文档链接|
|---|---|---|
| `create_collection()` | 创建 collection。 | [文档](200.collection/100.create-collection-of-api.md) |
| `get_collection()` | 获取指定 collection。 |[文档](200.collection/200.get-collection-of-api.md)|
| `get_or_create_collection()` | 创建或者查询 Collection，如果数据库中 Collection 不存在，则创建；如果存在 Collection，则获取相应的结果。 |[文档](200.collection/250.get-or-create-collection-of-api.md)|
| `list_collections()`  | 获取数据库 collection 列表。  |[文档](200.collection/300.list-collection-of-api.md)|
| `count_collection()`  | 统计数据库中的 Collection 数量  |[文档](200.collection/350.count-collection-of-api.md)|
| `delete_collection()`  | 删除指定 collection。|[文档](200.collection/400.delete-collection-of-api.md)|


### DML

:::info
仅支持在使用 `Client` 连接时，才能使用该接口。关于 `Client` 的详细介绍，参见 [Client](../50.apis/50.client.md)。
:::

| API 接口|描述     |相关文档链接|
|---|---|---|
| `add()` | 将新记录插入到 collection 中。 | [文档](300.dml/200.add-data-of-api.md) |
| `update()` | 更新 collection 中的现有记录。 |[文档](300.dml/300.update-data-of-api.md)|
| `upsert()`  | 插入新记录或更新现有记录。  |[文档](300.dml/400.upsert-data-of-api.md)|
| `delete()`  | 从 collection 中删除记录。|[文档](300.dml/500.delete-data-of-api.md)|

### DQL

:::info
仅支持在使用 `Client` 连接时，才能使用该接口。关于 `Client` 的详细介绍，参见 [Client](../50.apis/50.client.md)。
:::

| API 接口|描述     |相关文档链接|
|---|---|---|
| `query()` | 向量相似度搜索方法。 | [文档](400.dql/200.query-interfaces-of-api.md) |
| `get()` | 使用 ID、document 和 metadata（非矢量）从表中查询特定数据。 |[文档](400.dql/300.get-interfaces-of-api.md)|
| `hybrid_search()`  | 全文搜索和向量相似度搜索，采用排名法。  |[文档](400.dql/400.hybrid-search-of-api.md)|
