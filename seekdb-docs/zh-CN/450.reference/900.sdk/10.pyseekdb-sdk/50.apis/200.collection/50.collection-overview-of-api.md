# collection 管理

collection 在 pyseekdb 中为一种集合，类似与数据库中的表，用户可以进行创建、查询、删除对应的集合。

对于 collection 的操作，支持使用以下 API 接口。

| API 接口|描述     |相关文档链接|
|---|---|---|
| `create_collection()` | 创建 collection。 | [文档](100.create-collection-of-api.md) |
| `get_collection()` | 获取指定 collection。 |[文档](200.get-collection-of-api.md)|
| `get_or_create_collection()` | 创建或者查询 Collection，如果数据库中 Collection 不存在，则创建；如果存在 Collection，则获取相应的结果。 |[文档](250.get-or-create-collection-of-api.md)|
| `list_collections()`  | 获取数据库 collection 列表。  |[文档](300.list-collection-of-api.md)|
| `count_collection()`  | 统计数据库中的 Collection 数量  |[文档](350.count-collection-of-api.md)|
| `delete_collection()`  | 删除指定 collection。|[文档](400.delete-collection-of-api.md)|
