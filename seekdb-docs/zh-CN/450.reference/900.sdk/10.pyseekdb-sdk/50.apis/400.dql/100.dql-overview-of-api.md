# DQL 概述

DQL（数据查询语言）操作允许您使用各种查询方法从 collections 中检索数据。

对于 DQL 的操作，支持使用以下 API 接口。

| API 接口|描述     |相关文档链接|
|---|---|---|
| `query()` | 向量相似度搜索方法。 | [文档](200.query-interfaces-of-api.md) |
| `get()` | 使用 ID、document 和 metadata（非矢量）从表中查询特定数据。 |[文档](300.get-interfaces-of-api.md)|
| `hybrid_search()`  | 全文搜索和向量相似度搜索，采用排名法。  |[文档](400.hybrid-search-of-api.md)|
