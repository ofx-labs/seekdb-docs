# DBMS_PARTITION 概述

`DBMS_PARTITION` 系统包用于提供动态分区管理接口。

## 子程序概览

下表列出了 seekdb 所支持的 `DBMS_PARTITION` 子程序和简要描述。

|                                   子程序                                 |           描述          |
|--------------------------------------------------------------------------|------------------------|
| [MANAGE_DYNAMIC_PARTITION](200.manage_dynamic_partition.md)| 用于执行一次动态分区管理任务，为动态分区表预创建未来分区、删除过期分区。|
