# CANCEL_GATHER_STATS

`CANCEL_GATHER_STATS` 用于取消统计信息收集命令。

## 语法

```sql
DBMS_STATS.cancel_gather_stats (
      taskid         VARCHAR2
    );
```

## 参数解释

|    参数    |        解释         |
|----------|-----------------------|
|taskid|表示正在收集任务的 ID。<main id="notice" type='explain'><h4>说明</h4><p>您可以通过查询 <code>GV$OB_OPT_STAT_GATHER_MONITOR</code> 视图获取<code>taskid</code>，更多关于该视图的介绍请参考 <a href="https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000002015081">GV\$OB_OPT_STAT_GATHER_MONITOR</a>。</p></main>|

## 示例

```sql
CALL DBMS_STATS.CANCEL_GATHER_STATS(taskid);
```

  :::info

  其中 `taskid` 需要替换为实际值。

  :::
