# PURGE_LOG

`PURGE_LOG` 用于可以清除物化视图日志，从而减少日志文件的大小。清除物化视图日志会删除所有与该表相关的日志记录。

## 语法

```sql
PROCEDURE purge_log(
    IN     master_name            VARCHAR(65535));
```

## 参数解释

| **参数**           | **解释**                                                                                                                                              |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| master      | 需要清除数据的物化视图日志的主表名称
格式为 `schema.object or object`。                                                                         |
