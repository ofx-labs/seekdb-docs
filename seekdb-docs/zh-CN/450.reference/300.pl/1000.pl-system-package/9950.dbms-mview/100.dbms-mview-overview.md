# DBMS_MVIEW 概述

`DBMS_MVIEW` 系统包用于刷新物化视图和清除物化视图日志。

## DBMS_MVIEW 子程序概览

下表列出了 seekdb 所支持的 `DBMS_MVIEW` 子程序和简要描述。

| 子程序     | 描述                                      |
| --------- | ----------------------------------------- |
| [PURGE_LOG](200.purge-log.md)	| 用于可以清除物化视图日志，清除物化视图日志会删除所有与该表相关的日志记录。  |
| [REFRESH](300.refresh.md)	    | 刷新指定的一个或多个物化视图。                   |
