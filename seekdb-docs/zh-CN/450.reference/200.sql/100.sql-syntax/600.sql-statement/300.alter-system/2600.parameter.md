# PARAMETER

## 描述

* `ALTER SYSTEM [SET] parameter_name = expression` 语句用于修改配置项的值。

* `ALTER SYSTEM [RESET] parameter_name = expression` 语句用于重置配置项的值。

## 使用限制及注意事项

配置项的生效方式一般分为动态生效和重启生效两种，大部分配置项为动态生效方式，即不需要重启 seekdb 即可生效。修改配置项前，可通过 `SHOW PARAMETERS LIKE` 语句确认配置项的生效方式。查看配置项生效方式的详细操作，请参见 [配置项](../../../../../400.guides/500.manage/100.configuration-management/200.system-configuration-items/100.configuration-items-overview.md)。

## 权限要求

执行 `ALTER SYSTEM [SET | RESET] parameter_name = expression` 语句需要当前用户拥有 `ALTER SYSTEM` 权限。有关 seekdb 权限的详细介绍，请参见 [seekdb 的权限分类](../../../../../400.guides/500.manage/300.security/300.access-control/200.user-and-permission/100.permission-classification.md)。

## 语法

### 修改配置项的值

```sql
ALTER SYSTEM [SET] alter_system_set_parameter_action [, alter_system_set_parameter_action...]

alter_system_set_parameter_action:
parameter_name = expression [COMMENT [=] 'text']
```

#### 参数说明

|     **参数**    |     **描述**         |
|-----------------|-----------------------|
| parameter_name | 指定待修改的配置项名称。    |
| expression     | 指定修改后的配置项的值。   |
| COMMENT        | 添加关于本次修改的注释。该参数为可选，建议不要省略。   |

### 重置配置项的值

```sql
ALTER SYSTEM [RESET] alter_system_set_parameter_action [, alter_system_set_parameter_action...]

alter_system_set_parameter_action:
parameter_name = expression
```

#### 参数说明

|     **参数**    |     **描述**         |
|-----------------|-----------------------|
| parameter_name | 指定待重置的配置项名称。    |
| expression     | 指定需要重置的配置项。   |

## 示例

### 修改配置项

修改配置项 `connection_control_max_connection_delay` 的值。

  ```sql
  ALTER SYSTEM SET connection_control_max_connection_delay = 2147483646;
  ```

### 重置配置项

重置 `log_disk_utilization_threshold` 配置项的示例如下：

  ```sql
  ALTER SYSTEM RESET log_disk_utilization_threshold;
  ```
