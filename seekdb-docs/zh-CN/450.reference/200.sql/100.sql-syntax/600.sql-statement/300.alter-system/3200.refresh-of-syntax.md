# REFRESH

## 描述

该语句系列用来触发刷新操作，包括刷新各类缓存。

## 语法

```sql
alter_system_refresh_stmt:
  ALTER SYSTEM refresh_action;

refresh_action:
  | FLUSH KVCACHE
```

## 参数解释

|         **参数**       |         **描述**          |
|------------------------|----------------------------|
| FLUSH KVCACHE          | 清空可动态伸缩的内存 KVCache。 |

## 示例

刷新所有的 KVCache。

  ```shell
  ALTER SYSTEM FLUSH KVCACHE;
  Query OK, 0 rows affected
  ```
