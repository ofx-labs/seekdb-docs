# DROP CONCURRENT_LIMITING_RULE

## 描述

该语句用于删除限流规则。

## 权限要求

执行 `DROP CONCURRENT_LIMITING_RULE` 语句，需要当前用户拥有 DROP 权限。有关 seekdb 权限的详细介绍，参见 [seekdb 的权限分类](../../../../400.guides/500.manage/300.security/300.access-control/200.user-and-permission/100.permission-classification.md)。

## 语法

```sql
DROP CONCURRENT_LIMITING_RULE [ IF EXISTS ] `ccl_rule_name1` [, `ccl_rule_name2`, ...]
```

其中，`ccl_rule_name` 表示限流规则的名称。

## 示例

:::info
被删除的限流规则会立即失效，此时该规则下等待队列中的 SQL 语句全部会被正常执行。
:::

执行如下命令删除限流规则 `sql1_keywords_ccl_rule`，`sql2_keywords_ccl_rule`

```shell
DROP ONCURRENT_LIMITING_RULE `sql1_keywords_ccl_rule`，`sql2_keywords_ccl_rule`;
```
