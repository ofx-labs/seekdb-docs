# SET CATALOG

## 描述

该语句用于切换 Catalog。

:::info
seekdb 支持 `SET CATALOG` 语句。
:::

## 权限要求

执行 `SET CATALOG` 语句，需要当前用户拥有 `USE CATALOG` 权限。有关 seekdb 权限的详细介绍，参见 [seekdb 的权限分类](../../../../400.guides/500.manage/300.security/300.access-control/200.user-and-permission/100.permission-classification.md)。

## 语法

```sql
SET CATALOG {INTERNAL | external_catalog_name};
```

## 参数说明

|        **参数**       | **描述** |
|-----------------------|----------|
| INTERNAL              | 表示切换成系统内部 Catalog。|
| external_catalog_name | 指定要切换为外部数据目录的名称。|

:::info
可以使用 `use internal.test` 或 `use odps_catalog.default` 一次性完成切换 Catalog 和 数据库的操作。
:::

## 示例

* 切换为外部数据目录 `test_odps_catalog`。

  ```sql
  SET CATALOG test_odps_catalog;
  ```

* 切换成系统内部 Catalog。

  ```sql
  SET CATALOG INTERNAL;
  ```
