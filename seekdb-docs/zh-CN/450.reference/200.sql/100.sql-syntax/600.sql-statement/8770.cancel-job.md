# CANCEL JOB

用于取消异步任务。

:::tip
* 执行 `CANCEL JOB` 需要指定 `job_id`。
* 正在运行的任务会被终止，未开始运行的任务不会运行。
* 已经运行结束的任务，执行 `cancel job` 会报 `ERROR 4018 (HY000): Entry not exist` 错误。
* 取消任务后仍然可以查看 job 状态。
:::

## 语法

```sql
CANCEL JOB 'job_id';
```

## 示例

```shel
CANCEL JOB '2010011125899906847397';
3 rows in set (0.012 sec)
```
