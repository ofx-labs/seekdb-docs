# OPTIMIZE

## 描述

该语句用于控制开启一轮渐进合并。

渐进合并的轮数由表的 `progressive_merge_num` 决定，如果没有为表配置 `progressive_merge_num`，则会使用默认值。

## 语法

```sql
OPTIMIZE {
     TABLE table_list
};

table_list:
   table_name [,table_name ...]
```

## 参数解释

|     参数      |                    描述                     |
|-------------|-------------------------------------------|
| table_name  | 进行渐进合并的表名。                                |

## 示例

对 `t1` 表进行渐进合并。

```sql
OPTIMIZE TABLE t1;
Query OK, 0 rows affected (0.001 sec)
```
