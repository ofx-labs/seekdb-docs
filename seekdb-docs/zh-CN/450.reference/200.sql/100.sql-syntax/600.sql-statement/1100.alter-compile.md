# ALTER COMPILE

## 描述

该语句用于触发 procedure 或 function 的重新编译。

## 权限要求

执行 `ALTER COMPILE` 语句，需要当前用户拥有 `ALTER` 权限。有关 seekdb 权限的详细介绍，参见 [seekdb 的权限分类](../../../../400.guides/500.manage/300.security/300.access-control/200.user-and-permission/100.permission-classification.md)。

## 语法

```sql
ALTER {procedure｜function} name COMPILE [reuse settings]
```

## 参数解释

| **参数**                         |      **描述**     |
|------------|----|
| procedure |指定 `alter` 的类型为 `procedure`。|
| function  |指定 `alter` 的类型为 `function`。|
| name      |重新编译的具体存储过程或函数的名字。|
| reuse settings |可选参数，使用 `reuse settings` 子句时，使用与原先 PL 对象相同设置的参数编译。不使用时，用当前 session 中设置的参数进行编译。|

## 示例

```shell
ALTER procedure test_p COMPILE reuse settings;
Query OK, 0 rows affected (0.242 sec)
```
