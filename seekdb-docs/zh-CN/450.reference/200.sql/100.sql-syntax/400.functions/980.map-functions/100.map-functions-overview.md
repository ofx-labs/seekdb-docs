# 映射函数概述

映射函数是 seekdb 系统中提供的一组专门用于处理映射数据的函数。这些函数允许用户执行复杂的映射查询和分析。

以下是一些 seekdb 的映射函数及其用途的概述：

## 映射函数类型

| 映射函数类型                               | 描述                                                                |
|------------------------------------------|--------------------------------------------------------------------|
| [映射构造函数](200.map-constructor-functions.md) | 用于构建新的映射数据类型的函数。 |
| [映射判断函数](300.map-decision-functions.md) | 对输入的映射数据进行逻辑判断，并返回布尔值的函数。 |
| [映射操作函数](600.map-manipulation-functions.md)  | 用于对输入的映射数据进行基本操作，如提取 Key、提取 Value 等，并返回操作后的映射数据。 |

## 映射函数全览

seekdb 当前支持的所有映射函数如下所示：

| 函数类型 | 函数名称 | 描述 |
|---------|---------|------|
| [映射构造函数](200.map-constructor-functions.md) | map | 用于构建一个映射数据。 |
| [运算符](200.map-constructor-functions.md)  | {} | 用于构建一个映射数据。 |
| [运算符](300.map-decision-functions.md)  | =/!= | 用于判断两个映射数据是否相等。 |
| [映射操作函数](600.map-manipulation-functions.md) | map_keys | 获取映射数据中的 Key 列表。 |
| [映射操作函数](600.map-manipulation-functions.md) | map_values | 获取映射数据中的 Value 列表。 |
