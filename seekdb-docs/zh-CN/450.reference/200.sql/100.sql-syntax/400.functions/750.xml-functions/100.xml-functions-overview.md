# XML 函数概述

seekdb当前版本所支持的 XML 函数如下表所示。

| **函数名称** | **描述** |
| ------------ | -------- |
| [EXTRACTVALUE](300.querying-xmltype-data-functions/100.extractvalue-function.md) | 返回与 XPath 表达式匹配的元素的第一个文本节点的文本。 |
| [UPDATEXML](400.modify-xmltype-data-functions/100.updatexml-function.md) | 用于更新 XML 字符串中指定路径的节点内容。|
