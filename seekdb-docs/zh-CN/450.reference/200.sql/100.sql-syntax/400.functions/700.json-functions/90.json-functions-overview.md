# JSON 函数概述

seekdb下所支持的 JSON 函数如下表所示。

| **函数名称** | **描述** |
| --- | --- |
| [JSON_ARRAY](100.create-json-text-function/100.json-array.md) | 创建 JSON 数组。 |
| [JSON_ARRAYAGG](600.json-aggregate-function/100.json-arrayagg.md) |将结果集聚合为单个 JSON 数组。|
| [JSON_ARRAY_APPEND](300.modify-json-text-function/100.json-array-append.md)|将指定的值附加到 JSON 文档。|
| [JSON_APPEND](300.modify-json-text-function/120.json-append.md) | 将指定的值附加到 JSON 文档。|
| [JSON_ARRAY_INSERT](300.modify-json-text-function/200.json-array-insert.md) |将数组插入 JSON 文档。|
| [JSON_CONTAINS](200.search-json-text-function/100.json-contains.md) |检验指定 JSON 文档是否包含在目标 JSON 文档中。 |
| [JSON_CONTAINS_PATH](200.search-json-text-function/200.json-contains-path.md)|检验指定的路径中的数据是否包含于目标 JSON 文档中。|
| [JSON_DEPTH](400.return-json-text-property-function/100.json-depth.md)| 返回 JSON 文档的最大深度。|
| [JSON_EXTRACT](200.search-json-text-function/300.json-extract.md) |从 JSON 文档中返回数据。|
| [JSON_INSERT](300.modify-json-text-function/300.json-insert.md) |将数据插入到 JSON 文档。|
| [JSON_KEYS](200.search-json-text-function/400.json-keys.md) |从 JSON 文档中返回对应路径 Key 的数组。|
| [JSON_LENGTH](400.return-json-text-property-function/200.json-length.md) |返回 JSON 文档的长度。|
| [JSON_MERGE](300.modify-json-text-function/400.json-merge.md) |合并多个 JSON 文档。|
| [JSON_MERGE_PATCH](300.modify-json-text-function/500.json-merge-patch.md) |合并多个 JSON 文档，结果不保留具有重复键的成员。|
| [JSON_MERGE_PRESERVE](300.modify-json-text-function/600.json-merge-preserve.md) |合并多个 JSON 文档。|
| [JSON_OBJECT](100.create-json-text-function/200.json-object.md)| 创建 JSON 对象。 |
| [JSON_OVERLAPS](200.search-json-text-function/500.json-overlaps.md) |比较两个 JSON 文档。|
| [JSON_PRETTY](500.json-utility-function/100.json-pretty.md)|按照格式打印 JSON 文档。|
| [JSON_QUOTE](100.create-json-text-function/300.json-quote.md)| 引用 JSON 文档|
| [JSON_REMOVE](300.modify-json-text-function/700.json-remove.md) |从 JSON 文档中删除数据。|
| [JSON_REPLACE](300.modify-json-text-function/800.json-replace.md)|替换 JSON 文档中的替换 JSON 文档中的值。|
| [JSON_SEARCH](200.search-json-text-function/600.json-search.md) |返回 JSON 文档中值的路径。|
| [JSON_SET](300.modify-json-text-function/900.json-set.md) | 将数据插入到 JSON 文档。|
| [JSON_STORAGE_SIZE](500.json-utility-function/200.json-storage-size.md) |存储 JSON 文档的二进制字节数。|
| [JSON_TABLE](450.json-table-functions.md)|从 JSON 文档中提取数据并返回具有指定列的关系表。|
| [JSON_TYPE](400.return-json-text-property-function/300.json-type.md) |返回 JSON 值的类型。|
| [JSON_UNQUOTE](300.modify-json-text-function/1000.json-unquote.md) |取消引用 JSON 值。|
| [JSON_VALUE](200.search-json-text-function/700.json-value.md) |从指定路径中提取 JSON 文档中的值并返回。|
| [JSON_VALID](400.return-json-text-property-function/400.json-valid.md) |判断 JSON 值是否有效。|
| [JSON_OBJECTAGG](600.json-aggregate-function/200.json-objectagg.md)|返回包含键值对的 JSON 对象。|
| [MEMBER OF](200.search-json-text-function/800.member-of.md)|如果被检索的元素和 JSON 数组中的任意一个元素相同，则该函数返回 1，否则返回 0。|
| [JSON_SCHEMA_VALIDATION_REPORT](700.json-schema-function/200.json-schema-validation-report.md) | 如果 JSON 文档符合 JSON Schema 则返回 1，不符合则返回 0。|
| [JSON_SCHEMA_VALID](700.json-schema-function/100.json-schema-valid.md) | 返回 JSON Schema 的验证报告，验证报告为 JSON 格式。|
