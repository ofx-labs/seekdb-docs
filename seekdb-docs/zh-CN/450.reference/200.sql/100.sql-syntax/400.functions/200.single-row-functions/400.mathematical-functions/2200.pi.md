# PI

## 声明

```sql
PI()
```

## 说明

返回 `π` 的值（圆周率）。

## 示例

```sql
SELECT PI();
+----------+
| PI()     |
+----------+
| 3.141593 |
+----------+
1 row in set (0.000 sec)

SELECT ROUND(2*PI(),2);
+----------------+
| ROUND(2*PI(),2)|
+----------------+
| 6.28           |
+----------------+
1 row in set (0.001 sec)
```
