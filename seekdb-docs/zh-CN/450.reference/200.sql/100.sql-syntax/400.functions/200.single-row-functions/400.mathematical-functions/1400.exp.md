# EXP

## 声明

```sql
EXP(X)
```

## 说明

返回自然底数 e 的 `X` 次幂。

## 示例

```sql
SELECT EXP(2);
+------------------+
| EXP(2)           |
+------------------+
| 7.38905609893065 |
+------------------+
1 row in set (0.001 sec)

SELECT EXP(0);
+--------+
| EXP(0) |
+--------+
|      1 |
+--------+
1 row in set (0.001 sec)

SELECT EXP(-2);
+--------------------+
| EXP(-2)            |
+--------------------+
| 0.1353352832366127 |
+--------------------+
1 row in set (0.001 sec)
```
