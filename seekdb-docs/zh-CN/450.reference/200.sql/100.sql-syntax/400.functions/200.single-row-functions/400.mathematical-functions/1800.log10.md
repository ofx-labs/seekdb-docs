# LOG10

## 声明

```sql
LOG10(X)
```

## 说明

返回以 10 为底的 `X` 的指数。

## 示例

```sql
SELECT LOG10(1000);
+-------------+
| LOG10(1000) |
+-------------+
|           3 |
+-------------+
1 row in set (0.001 sec)

SELECT LOG10(-1000);
+--------------+
| LOG10(-1000) |
+--------------+
|         NULL |
+--------------+
1 row in set, 1 warning (0.000 sec)
```
