# TIME_TO_SEC

## 声明

```sql
TIME_TO_SEC(time)
```

## 说明

将时间类型参数 `time` 转换为秒。

## 示例

```sql
SELECT TIME_TO_SEC('22:23:00');
+-------------------------+
| TIME_TO_SEC('22:23:00') |
+-------------------------+
|                   80580 |
+-------------------------+
1 row in set (0.001 sec)

SELECT TIME_TO_SEC('00:39:38');
+-------------------------+
| TIME_TO_SEC('00:39:38') |
+-------------------------+
|                    2378 |
+-------------------------+
1 row in set (0.000 sec)
```
