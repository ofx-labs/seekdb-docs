# QUARTER

## 声明

```sql
QUARTER(date)
```

## 说明

返回指定日期的季度值，取值范围为 \[1,4\]。

## 示例

```sql
SELECT QUARTER(NOW()), NOW();
+----------------+---------------------+
| QUARTER(NOW()) | NOW()               |
+----------------+---------------------+
|              4 | 2025-12-17 17:19:38 |
+----------------+---------------------+
1 row in set (0.001 sec)
```
