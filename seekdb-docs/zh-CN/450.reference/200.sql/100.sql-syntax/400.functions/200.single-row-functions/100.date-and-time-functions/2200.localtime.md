# LOCALTIME

## 声明

```sql
LOCALTIME
LOCALTIME([scale])
```

## 说明

返回当前日期时间。`LOCALTIME` 和 `LOCALTIME()` 是 `NOW()` 的同义词。

`scale` 表示微秒部分的精度，有效值为 `0` 到 `6` 的整数，默认值为 `0`。

## 示例

```sql
SELECT LOCALTIME(0);
+---------------------+
| LOCALTIME(0)        |
+---------------------+
| 2025-12-17 16:54:37 |
+---------------------+
1 row in set (0.001 sec)

SELECT LOCALTIME(6);
+----------------------------+
| LOCALTIME(6)               |
+----------------------------+
| 2025-12-17 16:55:25.584336 |
+----------------------------+
1 row in set (0.000 sec)
```
