# SYSDATE

## 声明

```sql
SYSDATE(fsp)
```

## 说明

返回当前日期和时间，格式为 `%Y-%m-%d %H.%i.%s`。

`fsp` 表示秒精度，即返回值的小数位秒部分包含多少位数字。有效值为 0 到 6 的整数，默认值为 0。

## 示例

```sql
SELECT SYSDATE();
+---------------------+
| SYSDATE()           |
+---------------------+
| 2025-12-17 17:45:07 |
+---------------------+
1 row in set (0.001 sec)
```
