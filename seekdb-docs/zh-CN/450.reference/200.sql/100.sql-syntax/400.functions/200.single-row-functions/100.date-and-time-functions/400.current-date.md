# CURRENT_DATE

## 声明

```sql
CURRENT_DATE()
CURRENT_DATE
```

## 说明

返回当前日期，不含时间部分。该函数是 [CURDATE](../100.date-and-time-functions/300.curdate.md) 的别名。

## 示例

```javascript
SELECT CURRENT_DATE;
+--------------+
| CURRENT_DATE |
+--------------+
| 2025-12-17   |
+--------------+
1 row in set (0.001 sec)

SELECT CURRENT_DATE();
+----------------+
| CURRENT_DATE() |
+----------------+
| 2025-12-17     |
+----------------+
1 row in set (0.001 sec)
```
