# DAYOFMONTH

## 声明

```sql
DAYOFMONTH(date)
```

## 说明

返回日期或时间日期 `date` 在一个月中的位置。

## 示例

```sql
SELECT DAYOFMONTH('2008-12-31');
+--------------------------+
| DAYOFMONTH('2008-12-31') |
+--------------------------+
|                       31 |
+--------------------------+
1 row in set (0.001 sec)
```
