# CURRENT_TIMESTAMP

## 声明

```javascript
CURRENT_TIMESTAMP([scale])
```

## 说明

返回当前时区的日期时间。

`scale` 表示微秒部分精度，有效值为 `0` 到 `6` 的整数，默认值为 `0`。

## 示例

```javascript
SELECT CURRENT_TIMESTAMP(6);
+----------------------------+
| CURRENT_TIMESTAMP(6)       |
+----------------------------+
| 2025-12-17 16:13:31.520977 |
+----------------------------+
1 row in set (0.001 sec)
```
