# OCTET_LENGTH

## 声明

```sql
OCTET_LENGTH(str)
```

## 说明

`OCTET_LENGTH()` 是 `LENGTH()` 的同义词。

## 示例

```sql
SELECT OCTET_LENGTH('中国'), OCTET_LENGTH('hello');
+------------------------+-----------------------+
| OCTET_LENGTH('中国')   | OCTET_LENGTH('hello') |
+------------------------+-----------------------+
|                      6 |                     5 |
+------------------------+-----------------------+
1 row in set (0.001 sec)
```
