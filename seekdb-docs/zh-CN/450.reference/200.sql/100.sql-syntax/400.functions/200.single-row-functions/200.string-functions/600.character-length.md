# CHARACTER_LENGTH

## 声明

```sql
CHARACTER_LENGTH(str)
```

## 说明

`CHARACTER_LENGTH()` 是 `CHAR_LENGTH()` 的同义词。

## 示例

```sql
SELECT CHARACTER_LENGTH('中国'), LENGTH('中国'),CHARACTER_LENGTH('hello');
+----------------------------+------------------+---------------------------+
| CHARACTER_LENGTH('中国')   | LENGTH('中国')   | CHARACTER_LENGTH('hello') |
+----------------------------+------------------+---------------------------+
|                          2 |                6 |                         5 |
+----------------------------+------------------+---------------------------+
1 row in set (0.001 sec)
```
