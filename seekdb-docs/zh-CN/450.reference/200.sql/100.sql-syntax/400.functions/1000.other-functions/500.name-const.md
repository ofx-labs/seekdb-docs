# NAME_CONST

## 声明

```sql
NAME_CONST(name,value)
```

## 说明

该函数返回值定的值。参数 `value`应该是常量。

`NAME_CONST()` 使所生成结果集的列具有指定的名称。

## 示例

```sql
SELECT NAME_CONST('my_id', 101);
+-------+
| my_id |
+-------+
|   101 |
+-------+
1 row in set (0.001 sec)
```

您也可以通过设置简单的别名，获得与上述示例完全相同的结果，如下所示：

```sql
SELECT 101 AS my_id;
```
