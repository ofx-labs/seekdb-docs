# 锁函数概述

seekdb支持用于操作锁的函数，如下表所示。

| **函数名称** | **描述** |
|--------------|----------|
| [GET_LOCK](200.get-lock.md) | 获取命名锁。|
| [IS_FREE_LOCK](300.is-free-lock.md) | 指定的锁是否空闲。|
| [IS_USED_LOCK](400.is-used-lock.md) | 指定的锁是否正在使用。|
| [RELEASE_ALL_LOCKS](500.release-all-locks.md) | 释放当前会话所有持有的锁。|
| [RELEASE_LOCK](600.release-lock.md) | 释放指定的锁。|
