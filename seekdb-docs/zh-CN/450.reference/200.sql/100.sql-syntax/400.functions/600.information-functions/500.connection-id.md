# CONNECTION_ID

## 声明

```sql
CONNECTION_ID()
```

## 说明

该函数返回当前会话的 Client Session ID，该 ID 是会话在客户端中的唯一标识。

## 示例

获取当前会话的 Client Session ID。

```sql
SELECT CONNECTION_ID() AS Client_Session_ID;
```

返回结果如下：

```shell
+-------------------+
| Client_Session_ID |
+-------------------+
|        3221488032 |
+-------------------+
1 row in set (0.001 sec)
```
