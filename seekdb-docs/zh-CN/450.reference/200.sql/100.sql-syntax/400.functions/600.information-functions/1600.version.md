# VERSION

## 声明

```sql
VERSION()
```

## 说明

返回当前 seekdb 与 MySQL 兼容的版本号，字符集为 utf8mb4。

## 示例

```sql
SELECT VERSION();
+----------------------------------+
| VERSION()                        |
+----------------------------------+
| 5.7.25-OceanBase SeekDB-v1.0.0.0 |
+----------------------------------+
1 row in set (0.001 sec)
```
