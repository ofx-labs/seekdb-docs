# ICU_VERSION

## 声明

```sql
ICU_VERSION()
```

## 说明

该函数返回支持正则表达式操作的 ICU 库的版本。
seekdb 使用 International Components for Unicode (ICU) 实现对正则表达式的支持，ICU 提供完整的 Unicode 支持并且是多字节安全的。

## 示例

```sql
SELECT ICU_VERSION();
+---------------+
| ICU_VERSION() |
+---------------+
| 69.1          |
+---------------+
1 row in set (0.001 sec)
```
