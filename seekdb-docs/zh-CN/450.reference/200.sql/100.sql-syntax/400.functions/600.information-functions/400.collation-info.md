# COLLATION

## 声明

```sql
COLLATION(str)
```

## 说明

返回字符串 `str` 的字符序。

您可以通过 `SHOW COLLATION` 语句查看 seekdb 支持的字符序。

## 示例

```sql
SELECT COLLATION("oceanbase");
+------------------------+
| COLLATION("oceanbase") |
+------------------------+
| utf8mb4_general_ci     |
+------------------------+
1 row in set (0.001 sec)

SELECT COLLATION("oceanbase" COLLATE utf8mb4_bin);
+--------------------------------------------+
| COLLATION("oceanbase" COLLATE utf8mb4_bin) |
+--------------------------------------------+
| utf8mb4_bin                                |
+--------------------------------------------+
1 row in set (0.001 sec)
```
