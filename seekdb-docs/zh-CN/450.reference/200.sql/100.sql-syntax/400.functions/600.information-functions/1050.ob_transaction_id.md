# OB_TRANSACTION_ID

## 声明

```sql
OB_TRANSACTION_ID()
```

## 说明

* 当前会话处于活跃事务中，则返回所处的事务 ID 整数值，大于 0。
* 当前会话如果未处于活跃事务中，返回 0。

## 示例

查询当前会话处于活跃事务中的事务 ID。

1. 开启事务。

   ```shell
   BEGIN;
   ```

2. 查询当前事务 ID。

   ```shell
   SELECT OB_TRANSACTION_ID();
   ```

   返回结果如下：

   ```shell
   +---------------------+
   | OB_TRANSACTION_ID() |
   +---------------------+
   |            14565563 |
   +---------------------+
   1 row in set (0.001 sec)
   ```

3. 提交事务。

   ```shell
   COMMIT;
   ```

4. 查询当前未处于活跃事务中的事务 ID。

   ```shell
   SELECT OB_TRANSACTION_ID();
   ```

   返回结果如下：

   ```shell
   +---------------------+
   | OB_TRANSACTION_ID() |
   +---------------------+
   |                   0 |
   +---------------------+
   1 row in set (0.001 sec)
   ```
