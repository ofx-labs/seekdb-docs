# UNCOMPRESSED_LENGTH

## 声明

```sql
UNCOMPRESSED_LENGTH(str)
```

## 说明

返回压缩字符串在 [COMPRESS()](../500.encryption-and-compression-functions/300.compress.md) 函数压缩前的长度。

## 示例

```sql
SELECT UNCOMPRESSED_LENGTH(COMPRESS('abc'));
+--------------------------------------+
| UNCOMPRESSED_LENGTH(COMPRESS('abc')) |
+--------------------------------------+
|                                    3 |
+--------------------------------------+
1 row in set (0.001 sec)
```
