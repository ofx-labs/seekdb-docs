# UNCOMPRESS

## 声明

```sql
UNCOMPRESS(str)
```

## 说明

解压缩由 [COMPRESS()](../500.encryption-and-compression-functions/300.compress.md) 函数压缩的字符串。

## 示例

```sql
SELECT UNCOMPRESS(COMPRESS('abc'));
+-----------------------------+
| UNCOMPRESS(COMPRESS('abc')) |
+-----------------------------+
| abc                         |
+-----------------------------+
1 row in set (0.001 sec)
```
