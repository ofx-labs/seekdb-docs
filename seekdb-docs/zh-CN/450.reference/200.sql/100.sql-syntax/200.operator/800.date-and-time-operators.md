# 日期时间运算符

seekdb 支持日期时间运算符，其功能等同于指定的时间日期函数。

seekdb 当前版本所支持的日期时间运算符的详细信息如下表所示。

| 运算符 | 操作数 |                                                    含义                                                    |
|-----|-----|----------------------------------------------------------------------------------------------------------|
| `+` | 二元  | 将日期和时间相加，等同于 `DATE_ADD` 函数。请参见 [DATE_ADD](https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000002018184)。 |
| `-` | 二元  | 将日期和时间相减，等同于 `DATE_SUB` 函数。请参见 [DATE_SUB](https://www.oceanbase.com/docs/common-oceanbase-database-cn-1000000002018194)。 |

示例如下：

```javascript
SELECT '2008-12-31 23:59:59' + INTERVAL 1 SECOND;
+-------------------------------------------+
| '2008-12-31 23:59:59' + INTERVAL 1 SECOND |
+-------------------------------------------+
| 2009-01-01 00:00:00                       |
+-------------------------------------------+
1 row in set (0.001 sec)
```
