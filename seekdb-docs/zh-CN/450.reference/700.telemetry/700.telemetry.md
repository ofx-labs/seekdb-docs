# 遥测

seekdb 内置了遥测功能，默认开启，用于匿名采集产品使用信息。这些信息帮助我们了解用户如何使用 seekdb、识别常见问题，并持续改进产品功能和用户体验。

## 隐私保护

遥测功能会对敏感信息（如 IP 地址）进行脱敏(哈希)处理，仅采集必要的技术指标，不会采集任何业务数据或用户隐私信息。

## 管理遥测功能

可以通过环境变量 `TELEMETRY_ENABLED` 控制遥测功能的开启和关闭。

关闭遥测功能：

```bash
export TELEMETRY_ENABLED=false
```

关闭遥测功能后重新开启：

```bash
export TELEMETRY_ENABLED=true
```

:::tip
遥测功能默认开启，无需手动设置该环境变量。仅在需要关闭时设置为 false。
:::

## 遥测功能会采集哪些信息

遥测功能采集以下匿名化的技术信息：

* 版本与部署信息：seekdb 的版本、部署方式。

* 主机环境信息：操作系统版本、CPU 型号/核心数、内存容量。

* 资源规格信息：为 seekdb 配置的内存、数据盘、日志盘容量等。

## 查看采集信息

遥测数据会在本地保存一份副本，您可以在部署目录下的 `run/telemetry.json` 文件中查看 seekdb 采集的具体信息内容。
