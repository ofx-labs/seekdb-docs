# REMOVE_RULE

The `REMOVE_RULE` procedure is used to delete a custom rewrite rule.

## Syntax

```sql
PROCEDURE REMOVE_RULE (
    rule_name          VARCHAR(256));
```

## Parameters

| Parameter | Description |
| --- | --- |
| rule_name | The name of the rule. |

## Examples

```shell
CALL DBMS_UDR.REMOVE_RULE('rule1');
Query OK, 0 rows affected

SELECT DB_NAME, RULE_NAME, PATTERN, REPLACEMENT, NORMALIZED_PATTERN, STATUS, PATTERN_DIGEST
    FROM oceanbase.DBA_OB_USER_DEFINED_RULES;
Empty set
```
