# DBMS_PARTITION overview

The `DBMS_PARTITION` system package provides an interface for dynamic partition management.

## Subprograms

The following table lists the `DBMS_PARTITION` subprograms supported by seekdb and their brief descriptions.

|                                   Subprogram                                 |           Description          |
|--------------------------------------------------------------------------|------------------------|
| [MANAGE_DYNAMIC_PARTITION](200.manage_dynamic_partition.md)| Executes a dynamic partition management task to pre-create future partitions and drop expired partitions for the dynamic partition tables. |
