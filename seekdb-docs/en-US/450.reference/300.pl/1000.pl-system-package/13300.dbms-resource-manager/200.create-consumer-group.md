# CREATE_CONSUMER_GROUP

The `CREATE_CONSUMER_GROUP` procedure is used to create a resource consumer group.

## Syntax

```sql
DBMS_RESOURCE_MANAGER.CREATE_CONSUMER_GROUP (
    consumer_group  IN VARCHAR2,
    comment         IN VARCHAR2 DEFAULT NULL);
```

## Parameters

|       Parameter      |    Description     |
|----------------|-----------|
| consumer_group | The name of the resource consumer group. |
| comment        | User comment.     |

## Examples

```sql
CALL DBMS_RESOURCE_MANAGER.CREATE_CONSUMER_GROUP('group1','Transaction processing group');
Query OK, 0 rows affected

CALL DBMS_RESOURCE_MANAGER.CREATE_CONSUMER_GROUP('group2','O&M group');
Query OK, 0 rows affected
```
