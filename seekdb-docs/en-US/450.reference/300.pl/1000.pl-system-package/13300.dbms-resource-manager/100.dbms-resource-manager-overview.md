# Overview

The `DBMS_RESOURCE_MANAGER` package is used to manage resource plans, resource consumer groups, and resource plan directives.

## Privileges

The caller must have the `ADMINISTER_RESOURCE_MANAGER` system privilege to execute the procedures in the `DBMS_RESOURCE_MANAGER` package.

## Subprograms

The following table lists the `DBMS_RESOURCE_MANAGER` subprograms supported by seekdb and provides a brief description of each.

|                                   Subprogram                                 |           Description           |
|--------------------------------------------------------------------------|------------------------|
| [CREATE_CONSUMER_GROUP](../13300.dbms-resource-manager/200.create-consumer-group.md)      | Creates a resource consumer group.             |
| [CREATE_PLAN](../13300.dbms-resource-manager/300.create-plan.md)                | Creates a resource plan.                |
| [CREATE_PLAN_DIRECTIVE](../13300.dbms-resource-manager/400.create-plan-directive.md)      | Creates a resource plan directive.              |
| [DELETE_CONSUMER_GROUP](../13300.dbms-resource-manager/500.delete-consume-group.md)      | Deletes a resource consumer group.               |
| [DELETE_PLAN](../13300.dbms-resource-manager/600.delete-plan.md)                | Deletes a specified plan and all plan directives that it references.  |
| [DELETE_PLAN_DIRECTIVE](../13300.dbms-resource-manager/700.delete-plan-directive.md)      | Deletes a resource plan directive.              |
| [SET_CONSUMER_GROUP_MAPPING](../13300.dbms-resource-manager/800.set-consumer-group-mappingn-directive.md) | Adds, deletes, or modifies entries for login and runtime attribute mappings. |
| [UPDATE_PLAN_DIRECTIVE](../13300.dbms-resource-manager/900.update-plan-directive.md)      | Updates a resource plan directive.              |
