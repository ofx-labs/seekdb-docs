# PURGE_LOG

`PURGE_LOG` is used to clear a materialized view log, thereby reducing the size of the log file. Clearing a materialized view log deletes all log records associated with the table.

## Syntax

```sql
PROCEDURE purge_log(
    IN     master_name            VARCHAR(65535));
```

## Parameter Description

| **Parameter**           | **Description**                                                                                                                                              |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| master      | The name of the materialized view log whose data needs to be cleared.
The format is `schema.object or object`.                                                                         |
