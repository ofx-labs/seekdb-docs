# Overview of DBMS_VECTOR

The `DBMS_VECTOR` package is used to manage vector indexes.

## Overview of DBMS_VECTOR subprograms

The following table lists the `DBMS_VECTOR` subprograms supported by seekdb and their brief descriptions.

| **Subprogram** | **Description** |
| --- | --- |
| [REFRESH_INDEX](200.refresh-index.md) | Refreshes a vector index. |
| [REBUILD_INDEX](300.rebuild-index.md) | Rebuilds a vector index. |
| [INDEX_VECTOR_MEMORY_ESTIMATE](400.index-vector-memory-estimate.md) | Estimates the memory usage of a vector index. |
| [INDEX_VECTOR_MEMORY_ADVISOR](500.index-vector-memory-advisor.md) | Analyzes the memory usage of a vector index. |
