# SET_MVREF_STATS_PARAMS

`SET_MVREF_STATS_PARAMS` is used to configure the collection level and data retention period for collecting statistics during materialized view refreshes.

## Syntax

```sql
PROCEDURE set_mvref_stats_params(
    IN     mv_name                VARCHAR(65535),
    IN     collection_level       VARCHAR(65535)     DEFAULT NULL,
    IN     retention_period       INT                DEFAULT NULL);
```

## Parameters

| **Parameter**           | **Description**                                                                                                                                          |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| mv_name         | The name of the materialized view. <ul><li>If this parameter is set to NULL, the operation applies to all materialized views. </li></ul>                                              |
| collection_level | The collection level of statistics. The default value is TYPICAL. Valid values:<ul><li>NONE: Do not collect statistics.</li><li>TYPICAL: Collect basic refresh statistics.</li><li>ADVANCED: Collect detailed refresh statistics.</li><li>NULL: Use the default value, TYPICAL.</li></ul> |
| retention_period | The number of days for which statistics are retained. Valid value: 1 to 365000. Statistics are automatically deleted after the specified number of days. The default value is 31 days. <ul><li>If this parameter is set to NULL, the default value is used.</li><li>If this parameter is set to -1, statistics are permanently retained.</li></ul>                               |
