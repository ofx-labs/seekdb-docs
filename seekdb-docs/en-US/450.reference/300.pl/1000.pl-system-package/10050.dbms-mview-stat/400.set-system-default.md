# SET_SYSTEM_DEFAULT

`SET_SYSTEM_DEFAULT` sets the system default value for the refresh statistics parameter.

## Syntax

```sql
PROCEDURE set_system_default(
    IN     parameter_name         VARCHAR(65535),
    IN     value                  VARCHAR(65535));
```

## Parameters

| **Parameter**         | **Description**                                                                                         |
|------------------|--------------------------------------------------------------------------------------------------|
| parameter_name   | The name of the parameter, which can be set to one of the following values:<ul><li>COLLECTION_LEVEL: the level of statistics collection. The default value is TYPICAL.</li><li>RETENTION_PERIOD: the number of days to retain statistics. The default value is 31 days.</li></ul> |
| value | <p>When the parameter name is COLLECTION_LEVEL, the value can be set to one of the following values:<ul><li>NONE: do not collect statistics.</li><li>TYPICAL: collect basic refresh statistics.</li><li>ADVANCED: collect detailed refresh statistics.</li><li>NULL: use the system default value, that is, TYPICAL.</li></ul></p> <p>When the parameter name is RETENTION_PERIOD, the value specifies the number of days to retain statistics: 1 ~ 365000. <ul><li>NULL: use the system default value.</li><li>-1: retain statistics permanently.</li></ul> </p>|
