# RESET_PARAM_DEFAULTS

The `RESET_PARAM_DEFAULTS` procedure resets all parameter defaults to the values recommended by seekdb.

## Syntax

```sql
DBMS_STATS.RESET_PARAM_DEFAULTS;
```

## Examples

```sql
CALL DBMS_STATS.RESET_PARAM_DEFAULTS;
Query OK, 0 rows affected
```
