# RESET_GLOBAL_PREF_DEFAULTS

The `RESET_GLOBAL_PREF_DEFAULTS` procedure resets global Prefs to their default values.

## Syntax

```sql
DBMS_STATS.RESET_GLOBAL_PREF_DEFAULTS;
```

## Example

Reset all global Prefs to their default values.

```sql
CALL DBMS_STATS.RESET_GLOBAL_PREF_DEFAULTS();
Query OK, 0 rows affected
```
