# Overview

The `DBMS_WORKLOAD_REPOSITORY` package is used to manage the Automatic Workload Repository (AWR).

## Permissions

All subprograms in the `DBMS_WORKLOAD_REPOSITORY` package require the `EXECUTE` privilege.

## Subprograms

The following table lists the subprograms supported by seekdb and provides a brief description for each.

| **Subprogram** | **Description** |
| --- | --- |
| [ASH_REPORT](200.ash-report.md) | Displays the ASH report in text format. |
| [CREATE_SNAPSHOT](300.create-snapshot.md) | Triggers the generation of a snapshot. |
| [DROP_SNAPSHOT_RANGE](400.drop-snapshot-range.md) | Deletes a range of snapshots. |
| [MODIFY_SNAPSHOT_SETTINGS](500.modify-snapshot-settings.md) | Modifies the WORKLOAD_REPOSITORY configuration. |
