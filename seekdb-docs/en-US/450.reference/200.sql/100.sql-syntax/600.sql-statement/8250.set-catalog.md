# SET CATALOG

## Description

This statement is used to switch the catalog.

:::info
seekdb supports the `SET CATALOG` statement.
:::

## Privilege requirements

To execute the `SET CATALOG` statement, the current user must have the `USE CATALOG` privilege. For more information about the privileges in seekdb, see [Privilege types in seekdb](../../../../400.guides/500.manage/300.security/300.access-control/200.user-and-permission/100.permission-classification.md).

## Syntax

```sql
SET CATALOG {INTERNAL | external_catalog_name};
```

## Parameters

|        **Parameter**       | **Description** |
|-----------------------|----------|
| INTERNAL              | Switches to the internal catalog.|
| external_catalog_name | Specifies the name of the external data catalog to switch to.|

:::info
You can use `use internal.test` or `use odps_catalog.default` to switch to a catalog and database in one operation.
:::

## Examples

* Switch to the external data catalog `test_odps_catalog`.

  ```sql
  SET CATALOG test_odps_catalog;
  ```

* Switch to the internal catalog.

  ```sql
  SET CATALOG INTERNAL;
  ```
