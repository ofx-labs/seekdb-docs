# PURGE RECYCLEBIN

## Description

This statement is used to empty the recycle bin.

:::tip
`PURGE RECYCLEBIN` will delete all database objects in the recycle bin. Only the root user can execute this statement. Please be cautious when performing this operation.
:::

## Syntax

```sql
 PURGE RECYCLEBIN;
```

## Example

Empty the recycle bin.

```sql
PURGE RECYCLEBIN;
Query OK, 0 rows affected (0.231 sec)
```
