# PURGE

## Description

This statement is used to delete database objects from the recycle bin, including databases, tables, and indexes.

## Syntax

```sql
 PURGE {DATABASE | TABLE | INDEX} object_name;

object_name:
    database_name
  | table_name
  | index_name
```

## Parameters

|    **Parameter**     |           **Description**      |
|---------------|---------|
| database_name | The name of the database to be deleted in the recycle bin. You cannot directly specify the name of the database.                                                                            |
| table_name    | The name of the table to be deleted in the recycle bin. You cannot directly specify the name of the table.                                                                                |
| index_name    | The name of the index to be deleted in the recycle bin. You cannot directly specify the name of the index.                                                                              |
