# SET CHARSET

## Description

This statement is used to set the character set for the client and results.

## Syntax

```sql
SET {CHARSET | CHARACTER SET} {DEFAULT | charset_name};
```

## Parameters

|      Parameter      |      Description               |
|--------------|-----------------|
| charset_name | Specifies the character set for the client and results. Valid values: <ul><li> `utf8`  </li> <li> `utf8mb4`   </li> <li> `gbk`   </li> <li> `utf16`   </li> <li> `gb18030`  </li></ul>  |
| DEFAULT      | Uses the system default character set.      |

## Examples

Set the client character set to `gbk`.

```sql
SET CHARSET 'gbk';
Query OK, 0 rows affected
```
