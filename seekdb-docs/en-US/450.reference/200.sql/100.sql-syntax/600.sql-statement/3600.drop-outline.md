# DROP OUTLINE

## Description

This statement is used to drop an outline in seekdb.

## Syntax

```sql
DROP OUTLINE outline_name;
```

## Parameters

|    **Parameter**    |       **Description**       |
|--------------|--------------------|
| outline_name | The name of the outline to be dropped. |

## Examples

Drop the outline `ol_1`.

```sql
DROP OUTLINE ol_1;
```
