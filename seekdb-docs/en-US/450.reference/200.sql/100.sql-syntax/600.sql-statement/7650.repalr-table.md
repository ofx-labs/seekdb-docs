# REPAIR TABLE

This statement is used to repair a potentially damaged table.

## Syntax

```sql
REPAIR [NO_WRITE_TO_BINLOG | LOCAL]
    TABLE tbl_name [, tbl_name] ...
    [QUICK] [EXTENDED] [USE_FRM]
```

:::info
In the current version, only the syntax with the keyword is supported, and the feature is temporarily not available.
:::
