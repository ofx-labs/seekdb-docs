# ICU_VERSION

## Declaration

```sql
ICU_VERSION()
```

## Description

This function returns the version of the ICU library that supports regular expression operations.
seekdb uses International Components for Unicode (ICU) to implement support for regular expressions. ICU provides complete Unicode support and is safe for multibyte characters.

## Example

```sql
SELECT ICU_VERSION();
+---------------+
| ICU_VERSION() |
+---------------+
| 69.1          |
+---------------+
1 row in set (0.001 sec)
```
