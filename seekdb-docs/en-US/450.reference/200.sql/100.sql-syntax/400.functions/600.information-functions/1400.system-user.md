# SYSTEM_USER

## Declaration

```sql
SYSTEM_USER()
```

## Description

Returns the current username and hostname. This function is a synonym for `USER()`.

## Example

```sql
SELECT SYSTEM_USER();
+---------------------+
| SYSTEM_USER()       |
+---------------------+
| root@xxx.xx.xxx.xxx |
+---------------------+
1 row in set (0.001 sec)
```
