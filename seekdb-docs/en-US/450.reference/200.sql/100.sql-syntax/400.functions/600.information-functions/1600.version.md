# VERSION

## Syntax

```sql
VERSION()
```

## Description

Returns the version number of the current seekdb compatible with MySQL, in the utf8mb4 character set.

## Examples

```sql
SELECT VERSION();
+----------------------------------+
| VERSION()                        |
+----------------------------------+
| 5.7.25-OceanBase SeekDB-v1.0.0.0 |
+----------------------------------+
1 row in set (0.001 sec)
```
