# SESSION_USER

## Declaration

```sql
SESSION_USER()
```

## Description

Returns the username and hostname of the current user, using the utf8mb4 character set.

This function is a synonym for `USER()`.

## Examples

```sql
SELECT SESSION_USER();
+---------------------+
| SESSION_USER()      |
+---------------------+
| root@100.0.0.1 |
+---------------------+
1 row in set (0.001 sec)
```
