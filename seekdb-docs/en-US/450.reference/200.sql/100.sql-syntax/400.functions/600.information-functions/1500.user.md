# USER

## Declaration

```sql
USER()
```

## Description

Returns the username and hostname of the current user. The character set used for the return value is utf8mb4.

## Example

```sql
SELECT USER();
+---------------------+
| USER()              |
+---------------------+
| root@xxx.xx.xxx.xxx |
+---------------------+
1 row in set (0.001 sec)
```
