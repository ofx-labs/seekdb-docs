# RANDOM_BYTES

## Declaration

```sql
RANDOM_BYTES(len)
```

## Description

Returns a random binary string of length len, where len can be any value in the range [1, 1024].

## Example

```sql
SELECT RANDOM_BYTES(2);
+-----------------+
| RANDOM_BYTES(2) |
+-----------------+
| H              |
+-----------------+
1 row in set (0.001 sec)
```
