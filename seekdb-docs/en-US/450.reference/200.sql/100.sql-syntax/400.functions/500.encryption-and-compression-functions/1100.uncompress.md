# UNCOMPRESS

## Syntax

```sql
UNCOMPRESS(str)
```

## Description

Decompresses a string that was compressed by the [COMPRESS()](../500.encryption-and-compression-functions/300.compress.md) function.

## Examples

```sql
SELECT UNCOMPRESS(COMPRESS('abc'));
+-----------------------------+
| UNCOMPRESS(COMPRESS('abc')) |
+-----------------------------+
| abc                         |
+-----------------------------+
1 row in set (0.001 sec)
```
