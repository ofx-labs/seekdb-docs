# MD5

## Declaration

```javascript
MD5(str)
```

## Description

Returns the 128-bit MD5 checksum of a string.

The return value is a string of 32 hexadecimal digits. If `str` is `NULL`, `NULL` is returned.

## Example

```javascript
SELECT MD5(1);
+----------------------------------+
| MD5(1)                           |
+----------------------------------+
| c4ca4238a0b923820dcc509a6f75849b |
+----------------------------------+
1 row in set (0.001 sec)
```
