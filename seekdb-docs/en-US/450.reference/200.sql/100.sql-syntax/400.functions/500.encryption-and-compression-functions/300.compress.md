# COMPRESS

## Syntax

```sql
COMPRESS(str)
```

## Description

Compresses a string and returns the result as a binary string. You can use the [UNCOMPRESS](../500.encryption-and-compression-functions/1100.uncompress.md) function to decompress the result.

## Examples

```sql
SELECT COMPRESS('abc');
+-----------------+
| COMPRESS('abc') |
+-----------------+
|    x?KLJ M'  |
+-----------------+
1 row in set (0.001 sec)
```
