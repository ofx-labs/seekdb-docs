# UNCOMPRESSED_LENGTH

## Declaration

```sql
UNCOMPRESSED_LENGTH(str)
```

## Description

Returns the length of the string before it was compressed by the [COMPRESS()](../500.encryption-and-compression-functions/300.compress.md) function.

## Examples

```sql
SELECT UNCOMPRESSED_LENGTH(COMPRESS('abc'));
+--------------------------------------+
| UNCOMPRESSED_LENGTH(COMPRESS('abc')) |
+--------------------------------------+
|                                    3 |
+--------------------------------------+
1 row in set (0.001 sec)
```
