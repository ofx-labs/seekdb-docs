# SHA

## Declaration

```sql
SHA(str)
```

## Description

Computes and returns the 160-bit SHA-1 checksum of the given string as a string of 40 hexadecimal digits. Equivalent to the `SHA1()` function.

## Example

```sql
SELECT SHA('OB');
+------------------------------------------+
| SHA('OB')                                |
+------------------------------------------+
| 29a6de2fd8618f31afb281a608c57995b7eb105d |
+------------------------------------------+
1 row in set (0.001 sec)
```
