# XML functions overview

The following table lists the XML functions supported in the current version of seekdb.

| **Function** | **Description** |
| ------------ | -------- |
| [EXTRACTVALUE](300.querying-xmltype-data-functions/100.extractvalue-function.md) | Returns the text of the first text node that matches the XPath expression. |
| [UPDATEXML](400.modify-xmltype-data-functions/100.updatexml-function.md) | Updates the content of the node at the specified path in the XML string. |
