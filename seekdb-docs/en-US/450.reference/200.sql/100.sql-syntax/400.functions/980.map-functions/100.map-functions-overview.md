# Map functions overview

Map functions are a set of functions provided by the seekdb system for processing map data. These functions allow users to perform complex map queries and analyses.

Here is an overview of some map functions in seekdb and their purposes:

## Map function types

| Map function type                               | Description                                                                |
|------------------------------------------|--------------------------------------------------------------------|
| [Map constructor functions](200.map-constructor-functions.md) | Functions for constructing new map data types. |
| [Map decision functions](300.map-decision-functions.md) | Functions that perform logical judgments on input map data and return boolean values. |
| [Map manipulation functions](600.map-manipulation-functions.md)  | Functions for performing basic operations on input map data, such as extracting keys and values, and returning the modified map data. |

## Map functions overview

The following table lists all the map functions supported by seekdb:

| Function type | Function name | Description |
|---------|---------|------|
| [Map constructor functions](200.map-constructor-functions.md) | map | A function for constructing a map data. |
| [Operators](200.map-constructor-functions.md)  | {} | An operator for constructing a map data. |
| [Operators](300.map-decision-functions.md)  | =/!= | Operators for determining whether two map data are equal. |
| [Map manipulation functions](600.map-manipulation-functions.md) | map_keys | A function for obtaining the key list from map data. |
| [Map manipulation functions](600.map-manipulation-functions.md) | map_values | A function for obtaining the value list from map data. |
