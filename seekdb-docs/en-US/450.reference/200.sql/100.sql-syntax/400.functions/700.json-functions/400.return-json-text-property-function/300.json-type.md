# JSON_TYPE

## Description

This function returns a utf8mb4 string that represents the type of the JSON value.

## Syntax

```sql
JSON_TYPE(json_val)
```

## Description

The `json_val` parameter can be an object, an array, or a scalar.

If the parameter is `NULL`, the function returns `NULL`. If the parameter is not a valid JSON value, an error is returned.
