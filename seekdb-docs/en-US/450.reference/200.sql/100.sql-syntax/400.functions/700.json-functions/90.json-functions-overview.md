# JSON functions overview

The following table lists the JSON functions supported by seekdb.

| **Function** | **Description** |
| --- | --- |
| [JSON_ARRAY](100.create-json-text-function/100.json-array.md) | Creates a JSON array. |
| [JSON_ARRAYAGG](600.json-aggregate-function/100.json-arrayagg.md) | Aggregates results into a single JSON array. |
| [JSON_ARRAY_APPEND](300.modify-json-text-function/100.json-array-append.md)| Appends the specified value to a JSON document. |
| [JSON_APPEND](300.modify-json-text-function/120.json-append.md) | Appends the specified value to a JSON document. |
| [JSON_ARRAY_INSERT](300.modify-json-text-function/200.json-array-insert.md) | Inserts an array into a JSON document. |
| [JSON_CONTAINS](200.search-json-text-function/100.json-contains.md) | Checks whether the specified JSON document is contained in the target JSON document. |
| [JSON_CONTAINS_PATH](200.search-json-text-function/200.json-contains-path.md)| Checks whether the data in the specified path is contained in the target JSON document. |
| [JSON_DEPTH](400.return-json-text-property-function/100.json-depth.md)| Returns the maximum depth of a JSON document. |
| [JSON_EXTRACT](200.search-json-text-function/300.json-extract.md) | Returns data from a JSON document. |
| [JSON_INSERT](300.modify-json-text-function/300.json-insert.md) | Inserts data into a JSON document. |
| [JSON_KEYS](200.search-json-text-function/400.json-keys.md) | Returns an array of keys corresponding to the specified path in a JSON document. |
| [JSON_LENGTH](400.return-json-text-property-function/200.json-length.md) | Returns the length of a JSON document. |
| [JSON_MERGE](300.modify-json-text-function/400.json-merge.md) | Merges multiple JSON documents. |
| [JSON_MERGE_PATCH](300.modify-json-text-function/500.json-merge-patch.md) | Merges multiple JSON documents, and does not retain members with duplicate keys. |
| [JSON_MERGE_PRESERVE](300.modify-json-text-function/600.json-merge-preserve.md) | Merges multiple JSON documents. |
| [JSON_OBJECT](100.create-json-text-function/200.json-object.md)| Creates a JSON object. |
| [JSON_OVERLAPS](200.search-json-text-function/500.json-overlaps.md) | Compares two JSON documents. |
| [JSON_PRETTY](500.json-utility-function/100.json-pretty.md)| Prints a JSON document in a formatted manner. |
| [JSON_QUOTE](100.create-json-text-function/300.json-quote.md)| Quotes a JSON document. |
| [JSON_REMOVE](300.modify-json-text-function/700.json-remove.md) | Removes data from a JSON document. |
| [JSON_REPLACE](300.modify-json-text-function/800.json-replace.md)| Replaces values in a JSON document. |
| [JSON_SEARCH](200.search-json-text-function/600.json-search.md) | Returns the path of a value in a JSON document. |
| [JSON_SET](300.modify-json-text-function/900.json-set.md) | Inserts data into a JSON document. |
| [JSON_STORAGE_SIZE](500.json-utility-function/200.json-storage-size.md) | Returns the number of binary bytes stored in a JSON document. |
| [JSON_TABLE](450.json-table-functions.md)| Extracts data from a JSON document and returns a relational table with specified columns. |
| [JSON_TYPE](400.return-json-text-property-function/300.json-type.md) | Returns the type of a JSON value. |
| [JSON_UNQUOTE](300.modify-json-text-function/1000.json-unquote.md) | Unquotes a JSON value. |
| [JSON_VALUE](200.search-json-text-function/700.json-value.md) | Extracts and returns a value from a JSON document at the specified path. |
| [JSON_VALID](400.return-json-text-property-function/400.json-valid.md) | Checks whether a JSON value is valid. |
| [JSON_OBJECTAGG](600.json-aggregate-function/200.json-objectagg.md)| Returns a JSON object that contains key-value pairs. |
| [MEMBER OF](200.search-json-text-function/800.member-of.md)| Returns 1 if the retrieved element is the same as any element in the JSON array, otherwise returns 0. |
| [JSON_SCHEMA_VALIDATION_REPORT](700.json-schema-function/200.json-schema-validation-report.md) | Returns 1 if the JSON document conforms to the JSON schema, otherwise returns 0. |
| [JSON_SCHEMA_VALID](700.json-schema-function/100.json-schema-valid.md) | Returns a JSON schema validation report in JSON format. |
