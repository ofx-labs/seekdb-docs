# Overview of lock functions

seekdb supports the following lock functions:

| **Function** | **Description** |
|--------------|----------|
| [GET_LOCK](200.get-lock.md) | Gets a named lock.|
| [IS_FREE_LOCK](300.is-free-lock.md) | Checks whether the specified lock is free.|
| [IS_USED_LOCK](400.is-used-lock.md) | Checks whether the specified lock is in use.|
| [RELEASE_ALL_LOCKS](500.release-all-locks.md) | Releases all locks held by the current session.|
| [RELEASE_LOCK](600.release-lock.md) | Releases the specified lock.|
