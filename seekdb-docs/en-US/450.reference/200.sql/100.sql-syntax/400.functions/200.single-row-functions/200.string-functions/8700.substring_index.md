# SUBSTRING_INDEX

## Declaration

```sql
SUBSTRING_INDEX(str, delim, count)
```

## Description

Returns the substring from the beginning of the string `str` up to and excluding the `count`th occurrence of the delimiter `delim`.

* If `count` is positive, the substring from the beginning of the string `str` up to and excluding the `count`th occurrence of the delimiter `delim` is returned.

* If `count` is negative, the substring from the `count`th occurrence of the delimiter `delim` from the end of the string `str` is returned.

* If any parameter is `NULL`, `NULL` is returned.

* If `str` or `delim` is an empty string, an empty string is returned.

* If `count=0`, an empty string is returned.

* The parameters `str`, `delim`, and `count` support implicit conversion between numeric and string types.
