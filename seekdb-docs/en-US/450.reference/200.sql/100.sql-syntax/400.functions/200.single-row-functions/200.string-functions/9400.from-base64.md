# FROM_BASE64

## Declaration

```sql
FROM_BASE64(str)
```

## Description

Decodes a base-64 encoded string and returns it as a binary string. It is commonly used to decode strings encoded by the `TO_BASE64()` function. For more information, see [TO_BASE64()](../200.string-functions/9300.to-base64.md).

## Example

```sql
SELECT FROM_BASE64('T0I=');
+---------------------+
| FROM_BASE64('T0I=') |
+---------------------+
| OB                  |
+---------------------+
1 row in set (0.001 sec)
```
