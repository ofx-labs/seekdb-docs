# UCASE

## Statement

```sql
UCASE(str)
```

## Description

Converts lowercase letters in the `str` parameter to uppercase letters. This function is a synonym for the `UPPER()` function.

## Example

```sql
SELECT UCASE('obclient Hello!');
+----------------------------+
| UCASE('obclient hello!') |
+----------------------------+
| Hi OBCLIENT             |
+----------------------------+
1 row in set (0.001 sec)
```
