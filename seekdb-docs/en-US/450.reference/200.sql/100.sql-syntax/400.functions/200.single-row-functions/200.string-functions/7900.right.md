# RIGHT

## Declaration

```sql
RIGHT(str,len)
```

## Description

Returns the rightmost `len` characters of the string `str`. If `str` or `len` is `NULL`, it returns `NULL`.

This function is multibyte-safe.

## Examples

```sql
SELECT RIGHT('oceanbase',4);
+----------------------+
| RIGHT('oceanbase',4) |
+----------------------+
| base                 |
+----------------------+
1 row in set (0.001 sec)
```
