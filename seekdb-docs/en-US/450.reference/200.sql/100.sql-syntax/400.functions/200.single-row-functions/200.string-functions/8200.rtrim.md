# RTRIM

## Declaration

```sql
RTRIM(str)
```

## Description

Removes trailing spaces from the string `str`.

## Example

```sql
SELECT RTRIM('    obclient    ') AS rtrim;
+--------------+
| rtrim        |
+--------------+
|     obclient |
+--------------+
1 row in set (0.001 sec)
```
