# LTRIM

## Declaration

```sql
LTRIM(str)
```

## Description

Removes spaces from the left side of a string.

## Example

```sql
SELECT LTRIM('    obclient    ') AS ltrim;
+--------------+
| ltrim        |
+--------------+
| obclient     |
+--------------+
1 row in set (0.001 sec)
```
