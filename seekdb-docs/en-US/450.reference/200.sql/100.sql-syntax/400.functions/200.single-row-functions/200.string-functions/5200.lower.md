# LOWER

## Declaration

```sql
LOWER(str)
```

## Description

Convert the uppercase letters in a string to lowercase letters.

This function is the synonym of `LCASE()`.

## Example

```sql
SELECT LOWER('OBCLIENThello!');
+-----------------------------+
| LOWER('Hi OBCLIENT!') |
+-----------------------------+
| Hello obclient!          |
+-----------------------------+
1 row in set (0.001 sec)
```
