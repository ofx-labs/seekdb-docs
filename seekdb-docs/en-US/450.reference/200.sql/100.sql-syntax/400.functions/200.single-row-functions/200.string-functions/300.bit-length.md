# BIT_LENGTH

## Declaration

```sql
BIT_LENGTH(str)
```

## Description

Returns the bit length of the string `str`.

## Example

```sql
SELECT BIT_LENGTH('oceanbase');
+-------------------------+
| BIT_LENGTH('oceanbase') |
+-------------------------+
|                      72 |
+-------------------------+
1 row in set (0.001 sec)
```
