# UPPER

## Statements

```sql
UPPER(str)
```

## Notes

Convert lowercase letters in the string `str` to uppercase.

## Examples

```sql
SELECT UPPER('obclient hello!');
+----------------------------+
| upper('obclientHello!')  |
+----------------------------+
| Hello, OBCLIENT!           |
+----------------------------+
1 row in set (0.001 sec)
```
