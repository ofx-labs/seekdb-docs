# TO_BASE64

## Declaration

```sql
TO_BASE64(str)
```

## Description

Converts the string parameter to base-64 encoding. You can use the `FROM_BASE64` function to decode a base-64 encoded string. For more information, see [FROM_BASE64()](../200.string-functions/9400.from-base64.md).

## Example

```sql
SELECT TO_BASE64('OB');
+-----------------+
| TO_BASE64('OB')
+-----------------+
| T0I=            |
+-----------------+
1 row in set (0.000 sec)
```
