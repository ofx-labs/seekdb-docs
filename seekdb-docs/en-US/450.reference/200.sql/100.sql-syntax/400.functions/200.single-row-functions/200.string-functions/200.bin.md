# BIN

## Declaration

```sql
bin(N)
```

## Description

Returns the binary representation of the number `N`. This function is equivalent to `CONV(N,10,2)`.

If `N` is `NULL`, it returns `NULL`.

## Examples

```sql
SELECT BIN(12);
+---------+
| BIN(12) |
+---------+
| 1100    |
+---------+
1 row in set (0.001 sec)
```
