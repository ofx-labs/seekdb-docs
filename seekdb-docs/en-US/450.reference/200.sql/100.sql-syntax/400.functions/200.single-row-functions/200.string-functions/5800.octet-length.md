# OCTET_LENGTH

## Syntax

```sql
OCTET_LENGTH(str)
```

## Description

`OCTET_LENGTH()` is a synonym of `LENGTH()`.

## Examples

```sql
SELECT OCTET_LENGTH('China'), OCTET_LENGTH('hello');
+------------------------+-----------------------+
| OCTET_LENGTH('China')  | OCTET_LENGTH('hello') |
+------------------------+-----------------------+
|                      6 |                     5 |
+------------------------+-----------------------+
1 row in set (0.001 sec)
```
