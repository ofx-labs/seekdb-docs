# OCT

## Declaration

```sql
OCT(N)
```

## Description

Returns the octal value of `N`, which is equivalent to `CONV(N,10,8)`.

If `N` is `NULL`, the function returns `NULL`.

## Examples

```sql
SELECT OCT(12);
+---------+
| OCT(12) |
+---------+
| 14      |
+---------+
1 row in set (0.001 sec)
```
