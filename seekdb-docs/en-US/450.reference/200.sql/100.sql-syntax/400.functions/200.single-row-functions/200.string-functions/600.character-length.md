# CHARACTER_LENGTH

## Syntax

```sql
CHARACTER_LENGTH(str)
```

## Description

`CHARACTER_LENGTH()` is a synonym of `CHAR_LENGTH()`.

## Examples

```sql
SELECT CHARACTER_LENGTH('China'), LENGTH('China'),CHARACTER_LENGTH('hello');
+----------------------------+------------------+---------------------------+
| CHARACTER_LENGTH('China')  | LENGTH('China')  | CHARACTER_LENGTH('hello') |
+----------------------------+------------------+---------------------------+
|                          2 |                6 |                         5 |
+----------------------------+------------------+---------------------------+
1 row in set (0.001 sec)
```
