# SPACE

## Declaration

```sql
SPACE(N)
```

## Description

Returns a string that contains `N` spaces.

## Example

Returns a string that contains 5 spaces.

```sql
SELECT SPACE(5);
+----------+
| SPACE(5) |
+----------+
|          |
+----------+
1 row in set (0.000 sec)
```
