# LCASE

## Declaration

```sql
LCASE(str)
```

## Notes:

Convert uppercase letters in the string to lowercase.

The function is the synonym of `LOWER()`.

## Examples

```sql
SELECT LCASE('OBCLIENTHello!');
+----------------------------+
| LCASE('OBCLIENT Hi!')    |
+----------------------------+
| Hello, obclient!             |
+----------------------------+
1 row in set (0.001 sec)
```
