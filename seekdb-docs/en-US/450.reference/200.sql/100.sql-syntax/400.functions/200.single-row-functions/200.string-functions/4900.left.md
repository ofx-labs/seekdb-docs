# LEFT

## Syntax

```sql
LEFT(str,len)
```

## Description

Returns the leftmost `len` characters from the string `str`. If `str` or `len` is `NULL`, it returns `NULL`.

This function is multibyte-safe.

## Examples

```sql
SELECT LEFT('oceanbase', 5);
+----------------------+
| LEFT('oceanbase', 5) |
+----------------------+
| ocean                |
+----------------------+
1 row in set (0.001 sec)
```
