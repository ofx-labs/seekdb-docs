# REGEXP

## Declaration

```sql
str [NOT] REGEXP | RLIKE pat
```

## Description

Matches a regular expression. If any of the parameters is `NULL`, the result is `NULL`.

## Examples

```sql
SELECT
   1234 REGEXP 1,
   'hello' RLIKE 'h%'
   \G
*************************** 1. row ***************************
     1234 REGEXP 1: 1
'hello' RLIKE 'h%': 0
1 row in set (0.002 sec)
```
