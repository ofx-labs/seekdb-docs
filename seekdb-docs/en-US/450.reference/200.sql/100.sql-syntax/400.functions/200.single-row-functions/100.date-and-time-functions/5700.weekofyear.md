# WEEKOFYEAR

## Declaration

```sql
WEEKOFYEAR(date)
```

## Description

Returns the position of the week specified by the `date` date or time type in the year.

## Example

```sql
SELECT WEEKOFYEAR('2020-07-16');
+--------------------------+
| WEEKOFYEAR('2020-07-16') |
+--------------------------+
|                       29 |
+--------------------------+
1 row in set (0.001 sec)
```
