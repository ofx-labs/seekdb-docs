# HOUR

## Declaration

```sql
HOUR(time)
```

## Description

Returns the numeric value of the hour part of the `time` type.

## Example

```sql
SELECT HOUR('10:05:03');
+------------------+
| HOUR('10:05:03') |
+------------------+
|               10 |
+------------------+
1 row in set (0.001 sec)

SELECT HOUR('272:59:59');
+-------------------+
| HOUR('272:59:59') |
+-------------------+
|               272 |
+-------------------+
1 row in set (0.000 sec)
```
