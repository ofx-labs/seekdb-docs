# DAYNAME

## Declaration

```sql
DAYNAME(date)
```

## Description

Returns the name of the weekday corresponding to the date.

## Example

```sql
SELECT DAYNAME('2022-09-08');
+-----------------------+
| DAYNAME('2022-09-08') |
+-----------------------+
| Thursday              |
+-----------------------+
1 row in set (0.001 sec)
```
