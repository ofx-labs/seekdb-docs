# DAYOFMONTH

## Declaration

```sql
DAYOFMONTH(date)
```

## Description

Returns the position of the date or datetime `date` in the month.

## Example

```sql
SELECT DAYOFMONTH('2008-12-31');
+--------------------------+
| DAYOFMONTH('2008-12-31') |
+--------------------------+
|                       31 |
+--------------------------+
1 row in set (0.001 sec)
```
