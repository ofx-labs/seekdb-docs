# DAYOFYEAR

## Declaration

```sql
DAYOFYEAR(date)
```

## Description

Returns the position of the date or datetime `date` in the year.

## Example

```sql
SELECT DAYOFYEAR('2021-07-15');
+-------------------------+
| DAYOFYEAR('2021-07-15') |
+-------------------------+
|                     196 |
+-------------------------+
1 row in set (0.001 sec)
```
