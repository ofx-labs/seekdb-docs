# TIME

## Declaration

```javascript
TIME(datetime)
```

## Description

Returns the time information of `datetime` as the `TIME` type.

## Example

```javascript
SELECT TIME('2003-12-31 01:02:03');
+-----------------------------+
| TIME('2003-12-31 01:02:03') |
+-----------------------------+
| 01:02:03.000000             |
+-----------------------------+
1 row in set (0.000 sec)
```
