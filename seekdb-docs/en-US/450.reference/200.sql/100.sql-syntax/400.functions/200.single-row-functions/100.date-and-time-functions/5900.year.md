# YEAR

## Declaration

```sql
YEAR(date)
```

## Description

Returns the year information of the `date` value.

## Example

```sql
SELECT YEAR('1987-01-01');
+--------------------+
| YEAR('1987-01-01') |
+--------------------+
|               1987 |
+--------------------+
1 row in set (0.001 sec)
```
