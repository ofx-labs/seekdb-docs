# TIMESTAMPADD

## Declaration

```javascript
TIMESTAMPADD(unit, interval_expr, date)
```

## Description

Performs arithmetic operations on date and time values.

The function is similar to [DATE_ADD](../100.date-and-time-functions/900.date-add.md), but the `unit` parameter can only be one of the following units:

* `MICROSECOND`

* `SECOND`

* `MINUTE`

* `HOUR`

* `DAY`

* `WEEK`

* `MONTH`

* `QUARTER`

* `YEAR`

## Examples

```javascript
SELECT
  TIMESTAMPADD(DAY, -5, '2010-01-01 00:00:00'),
  DATE_ADD('2010-01-01 00:00:00', INTERVAL -5 DAY)
     \G
*************************** 1. row ***************************
    TIMESTAMPADD(DAY, -5, '2010-01-01 00:00:00'): 2009-12-27 00:00:00
DATE_ADD('2010-01-01 00:00:00', INTERVAL -5 DAY): 2009-12-27 00:00:00
1 row in set (0.001 sec)
```
