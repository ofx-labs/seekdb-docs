# MONTH

## Declaration

```javascript
MONTH(date)
```

## Description

Returns the month information of the `date`.

## Example

```javascript
SELECT MONTH('2008-02-03');
+---------------------+
| MONTH('2008-02-03') |
+---------------------+
|                   2 |
+---------------------+
1 row in set (0.001 sec)
```
