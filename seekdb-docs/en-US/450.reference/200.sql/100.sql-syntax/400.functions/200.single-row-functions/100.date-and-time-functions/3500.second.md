# SECOND

## Declaration

```sql
SECOND(time)
```

## Description

Returns the number of seconds in the time `time`.

## Example

```sql
SELECT SECOND('18:20:17');
+--------------------+
| SECOND('18:20:17') |
+--------------------+
|                 17 |
+--------------------+
1 row in set (0.001 sec)
```
