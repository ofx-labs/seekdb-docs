# MONTHNAME

## Declaration

```sql
MONTHNAME(date)
```

## Description

Returns the full name of the month for the specified date `date`.

## Example

```sql
SELECT MONTHNAME('2008-02-03');
+-------------------------+
| MONTHNAME('2008-02-03') |
+-------------------------+
| February                |
+-------------------------+
1 row in set (0.001 sec)
```
