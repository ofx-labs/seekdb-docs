# QUARTER

## Declaration

```sql
QUARTER(date)
```

## Description

Returns the quarter value of a specified date, which ranges from \[1,4\].

## Example

```sql
SELECT QUARTER(NOW()), NOW();
+----------------+---------------------+
| QUARTER(NOW()) | NOW()               |
+----------------+---------------------+
|              4 | 2025-12-17 17:19:38 |
+----------------+---------------------+
1 row in set (0.001 sec)
```
