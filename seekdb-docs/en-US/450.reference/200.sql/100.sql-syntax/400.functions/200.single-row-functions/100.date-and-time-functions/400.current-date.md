# CURRENT_DATE

## Syntax

```sql
CURRENT_DATE()
CURRENT_DATE
```

## Description

Returns the current date without the time part. This function is an alias of [CURDATE](../100.date-and-time-functions/300.curdate.md).

## Examples

```javascript
SELECT CURRENT_DATE;
+--------------+
| CURRENT_DATE |
+--------------+
| 2025-12-17   |
+--------------+
1 row in set (0.001 sec)

SELECT CURRENT_DATE();
+----------------+
| CURRENT_DATE() |
+----------------+
| 2025-12-17     |
+----------------+
1 row in set (0.001 sec)
```
