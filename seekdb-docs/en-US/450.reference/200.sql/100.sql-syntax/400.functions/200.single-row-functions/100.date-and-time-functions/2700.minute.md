# MINUTE

## Declaration

```sql
MINUTE(time)
```

## Description

Returns the minute number in the time `time`.

## Example

```sql
SELECT MINUTE('18:20:17');
+--------------------+
| MINUTE('18:20:17') |
+--------------------+
|                 20 |
+--------------------+
1 row in set (0.001 sec)
```
