# DAY

## Declaration

```sql
DAY(date)
```

## Description

Returns the day of the month in the `date` value, ranging from 1 to 31. `DAY()` is an alias for `DAYOFMONTH()`.

## Example

```sql
SELECT DAY('2008-12-31');
+--------------------------+
| DAY('2008-12-31') |
+--------------------------+
|                       31 |
+--------------------------+
1 row in set (0.001 sec)
```
