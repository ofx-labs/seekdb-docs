# WEEKDAY

## Declaration

```sql
WEEKDAY(date)
```

## Description

Returns the position of a `date` value in the week.

## Example

```sql
SELECT WEEKDAY('2021-07-16 00:00:00');
+--------------------------------+
| WEEKDAY('2020-07-16 00:00:00') |
+--------------------------------+
|                              5 |
+--------------------------------+
1 row in set (0.001 sec)
```
