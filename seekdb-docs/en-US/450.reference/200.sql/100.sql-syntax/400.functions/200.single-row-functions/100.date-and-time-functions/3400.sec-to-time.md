# SEC_TO_TIME

## Declaration

```sql
SEC_TO_TIME(seconds)
```

## Description

Converts the `seconds` parameter to a time value. The result is limited to the range of the `TIME` data type.

## Example

```sql
SELECT SEC_TO_TIME(2378);
+-------------------+
| SEC_TO_TIME(2378) |
+-------------------+
| 00:39:38          |
+-------------------+
1 row in set (0.001 sec)

SELECT SEC_TO_TIME(2378) + 0;
+-----------------------+
| SEC_TO_TIME(2378) + 0 |
+-----------------------+
|                  3938 |
+-----------------------+
1 row in set (0.001 sec)
```
