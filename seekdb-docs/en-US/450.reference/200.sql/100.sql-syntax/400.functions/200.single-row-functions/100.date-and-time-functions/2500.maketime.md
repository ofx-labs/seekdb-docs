# MAKETIME

## Declaration

```sql
MAKETIME(hour,minute,second)
```

## Description

Returns the time value specified by `hour:minute:second`, where `second` can have a decimal part.

## Example

```sql
SELECT MAKETIME(12,15,30);
+--------------------+
| MAKETIME(12,15,30) |
+--------------------+
| 12:15:30.000000    |
+--------------------+
1 row in set (0.001 sec)
```
