# UTC_DATE

## Declaration

```sql
UTC_DATE()
```

## Description

Returns the current UTC date in the `%Y-%m-%d` format.

## Example

```sql
SELECT UTC_DATE(), UTC_DATE() + 0, UTC_DATE() + 6;
+------------+----------------+----------------+
| UTC_DATE() | UTC_DATE() + 0 | UTC_DATE() + 6 |
+------------+----------------+----------------+
| 2025-12-18 |       20251218 |       20251224 |
+------------+----------------+----------------+
1 row in set (0.001 sec)
```
