# DATE

## Declaration

```sql
DATE(expr)
```

## Description

Returns the date part of the time or time date expression `expr`.

## Example

```sql
SELECT DATE('2021-07-14 19:12:03');
+-----------------------------+
| DATE('2021-07-14 19:12:03') |
+-----------------------------+
| 2021-07-14                  |
+-----------------------------+
1 row in set (0.001 sec)
```
