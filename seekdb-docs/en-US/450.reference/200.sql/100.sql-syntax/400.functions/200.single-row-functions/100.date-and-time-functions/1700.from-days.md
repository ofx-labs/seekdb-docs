# FROM_DAYS

## Syntax

```javascript
FROM_DAYS(N)
```

## Description

Returns the `DATE` value that is `N` days from `0000-01-01`.

## Examples

```javascript
SELECT FROM_DAYS(736271), FROM_DAYS(700000);
+-------------------+-------------------+
| FROM_DAYS(736271) | FROM_DAYS(700000) |
+-------------------+-------------------+
| 2015-11-04        | 1916-07-15        |
+-------------------+-------------------+
1 row in set (0.001 sec)
```
