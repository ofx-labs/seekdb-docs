# TAN

## Declaration

```sql
TAN(X)
```

## Description

Returns the tangent of the given radians X.

## Example

```sql
SELECT TAN(PI());
+-------------------------+
| TAN(PI())               |
+-------------------------+
| -1.2246467991473532e-16 |
+-------------------------+
1 row in set (0.001 sec)
```
