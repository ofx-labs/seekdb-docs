# DEGREES

## Syntax

```sql
DEGREES(x)
```

## Description

The `DEGREES()` function converts the parameter `x` from radians to degrees and returns the result.

## Examples

The following examples convert the values returned by `PI()` and `PI()/2` to degrees.

```sql
SELECT DEGREES(PI());
+---------------+
| DEGREES(PI()) |
+---------------+
|           180 |
+---------------+
1 row in set (0.001 sec)

SELECT DEGREES(PI()/2);
+-------------------+
| DEGREES(PI()/2) |
+-------------------+
|                90 |
+-------------------+
1 row in set (0.001 sec)
```
