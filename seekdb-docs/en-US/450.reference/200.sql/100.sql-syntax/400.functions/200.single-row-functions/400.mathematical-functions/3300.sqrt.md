# SQRT

## Declaration

```sql
SQRT(X)
```

## Description

Returns the square root of `X`.

## Example

```sql
SELECT SQRT(4);
+---------+
| SQRT(4) |
+---------+
|       2 |
+---------+
1 row in set (0.001 sec)

SELECT SQRT(5);
+------------------+
| SQRT(5)          |
+------------------+
| 2.23606797749979 |
+------------------+
1 row in set (0.001 sec)

SELECT SQRT(-16);
+-----------+
| SQRT(-16) |
+-----------+
|      NULL |
+-----------+
1 row in set (0.001 sec)
```
