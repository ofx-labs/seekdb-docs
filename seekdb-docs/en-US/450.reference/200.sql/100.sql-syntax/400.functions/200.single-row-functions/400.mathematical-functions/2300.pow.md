# POW

## Declaration

```sql
POW(X,Y)
```

## Description

Returns the value of `X` raised to the power of `Y`.

If either `X` or `Y` is `NULL`, the function returns `NULL`.

## Example

```sql
SELECT POW(4,2), POW(4,-2), POW(1,NULL);
+----------+-----------+-------------+
| POW(4,2) | POW(4,-2) | POW(1,NULL) |
+----------+-----------+-------------+
|       16 |    0.0625 |        NULL |
+----------+-----------+-------------+
1 row in set (0.001 sec)
```
