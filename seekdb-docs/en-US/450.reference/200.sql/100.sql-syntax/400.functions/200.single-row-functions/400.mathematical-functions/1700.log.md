# LOG

## Declaration

```sql
LOG([B,]X)
```

## Description

`LOG(X)` returns the natural logarithm of `X`, which is the logarithm of `X` with base `e`. If `X` is less than or equal to `0`, it returns `NULL` and raises a warning `Invalid argument for logarithm`. When only `X` is provided, this function is equivalent to the `LN(X)` function.

`LOG(B,X)` returns the logarithm of `X` with base `B`. If `X` is less than or equal to `0` or `B` is less than or equal to `1`, it returns `NULL`.

## Examples

```sql
SELECT LOG(-10), LOG(0), LOG(10), LOG(-3.14), LOG(3.14);
+----------+--------+-------------------+------------+-------------------+
| LOG(-10) | LOG(0) | LOG(10)           | LOG(-3.14) | LOG(3.14)         |
+----------+--------+-------------------+------------+-------------------+
|     NULL |   NULL | 2.302585092994046 |       NULL | 1.144222799920162 |
+----------+--------+-------------------+------------+-------------------+
1 row in set, 3 warnings (0.001 sec)

SELECT LOG(10, 100), LOG(1,100);
+--------------+------------+
| LOG(10, 100) | LOG(1,100) |
+--------------+------------+
|            2 |       NULL |
+--------------+------------+
1 row in set, 1 warning (0.001 sec)
```
