# SIN

## Declaration

```sql
SIN(X)
```

## Description

Returns the sine of the given radians `X`.

## Example

```sql
SELECT SIN(PI());
+------------------------+
| SIN(PI())              |
+------------------------+
| 1.2246467991473532e-16 |
+------------------------+
1 row in set (0.001 sec)
```
