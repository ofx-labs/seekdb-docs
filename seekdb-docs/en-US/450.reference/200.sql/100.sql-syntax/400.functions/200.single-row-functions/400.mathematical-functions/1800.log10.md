# LOG10

## Declaration

```sql
LOG10(X)
```

## Description

Returns the base-10 logarithm of `X`.

## Example

```sql
SELECT LOG10(1000);
+-------------+
| LOG10(1000) |
+-------------+
|           3 |
+-------------+
1 row in set (0.001 sec)

SELECT LOG10(-1000);
+--------------+
| LOG10(-1000) |
+--------------+
|         NULL |
+--------------+
1 row in set, 1 warning (0.000 sec)
```
