# ASIN

## Declaration

```sql
ASIN(X)
```

## Description

Returns the inverse sine of `X`, which is the value whose sine is `X`. If `X` is not in the range `[-1, 1]`, the function returns `NULL`.

## Examples

```sql
SELECT ASIN(-10), ASIN(0), ASIN(10), ASIN(-0.5), ASIN(0.5);
+-----------+---------+----------+---------------------+--------------------+
| ASIN(-10) | ASIN(0) | ASIN(10) | ASIN(-0.5)          | ASIN(0.5)          |
+-----------+---------+----------+---------------------+--------------------+
|      NULL |       0 |     NULL | -0.5235987755982989 | 0.5235987755982989 |
+-----------+---------+----------+---------------------+--------------------+
1 row in set (0.001 sec)
```
