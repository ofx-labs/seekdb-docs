# ATAN

## Syntax

```sql
ATAN(X)
```

## Description

Returns the arctangent of `X`, that is, the value whose tangent is `X`.

## Example

```sql
SELECT ATAN(-10), ATAN(0), ATAN(10), ATAN(-0.5), ATAN(0.5);
+---------------------+---------+--------------------+---------------------+--------------------+
| ATAN(-10)           | ATAN(0) | ATAN(10)           | ATAN(-0.5)          | ATAN(0.5)          |
+---------------------+---------+--------------------+---------------------+--------------------+
| -1.4711276743037347 |       0 | 1.4711276743037347 | -0.4636476090008061 | 0.4636476090008061 |
+---------------------+---------+--------------------+---------------------+--------------------+
1 row in set (0.001 sec)
```
