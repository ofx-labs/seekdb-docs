# COT

## Declaration

```sql
COT(X)
```

## Description

Returns the cotangent of `X`.

## Example

```sql
SELECT COT(-10), COT(10), COT(-0.5), COT(0.5);
+---------------------+--------------------+--------------------+-------------------+
| COT(-10)            | COT(10)            | COT(-0.5)          | COT(0.5)          |
+---------------------+--------------------+--------------------+-------------------+
| -1.5423510453569202 | 1.5423510453569202 | -1.830487721712452 | 1.830487721712452 |
+---------------------+--------------------+--------------------+-------------------+
1 row in set (0.001 sec)
```
