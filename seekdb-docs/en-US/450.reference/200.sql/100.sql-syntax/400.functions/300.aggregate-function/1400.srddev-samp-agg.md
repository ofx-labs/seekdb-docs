# STDDEV_SAMP

## Declaration

```sql
STDDEV_SAMP(expr)
```

## Description

Returns the sample standard deviation of `expr`, which is the square root of the result of the `VAR_SAMP()` function. For more information about the `VAR_SAMP()` function, see [VAR_SAMP](../300.aggregate-function/1700.var-samp-agg.md).

If no rows match the condition, the function returns `NULL`.
