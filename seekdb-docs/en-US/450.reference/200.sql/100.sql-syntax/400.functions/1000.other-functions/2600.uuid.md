# UUID

## Declaration

```sql
uuid()
```

## Description

Generates a globally unique ID.

## Example

```sql
SELECT UUID();
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| eef95d17-e944-11eb-824a-ca1fa9040697 |
+--------------------------------------+
1 row in set (0.001 sec)
```
