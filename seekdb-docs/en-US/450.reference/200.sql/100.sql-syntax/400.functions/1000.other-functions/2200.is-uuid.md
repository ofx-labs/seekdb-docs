# IS_UUID

## Syntax

```sql
IS_UUID(string_uuid)
```

## Description

Returns 1 if the parameter is a valid UUID string, 0 if it is not, and `NULL` if the parameter is `NULL`.
A valid UUID string is one that can be parsed. This means it has the correct length and contains only allowed characters (hexadecimal digits and optional hyphens and braces). The most common format is:

```sql
aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee
```

The following format is also allowed:

```sql
aaaaaaaabbbbccccddddeeeeeeeeeeee
{aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee}
```
